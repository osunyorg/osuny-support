Comment ajouter un chiffre clé ?

body: >-

Ajouter un bloc

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

Choisir le bloc Chiffres Clés
Remplir le bloc

L'utilisateur peut ajouter des chiffres clés avec une jolie animation. Chaque chiffre peut être doté d'une unité et d'un texte. L'utilisateur rentre les champs suivants pour ajouter un bloc chiffre clé :
Titre
Publié ou non
Texte
Ajouter un ou plusieurs chiffre clé : numéro / unité / description
Possibilité de retirer un ou plusieurs chiffres clés

Exemples

Vous trouverez différents types d'exemples de mise en page de Chiffres Clés sur Example.osuny.org
