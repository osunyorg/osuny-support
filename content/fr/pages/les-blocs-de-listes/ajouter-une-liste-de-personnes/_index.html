Comment ajouter une liste de personnes ?

body: >-

Ajouter un bloc

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

Choisir le bloc personnes
Remplir le bloc

L'utilisateur peut ajouter une liste de personnes. L'utilisateur.rice rentre les champs suivants :
Titre
Publié ou non
Description
Ajouter
une ou plusieurs personnes : choix via une liste déroulante des
personnes avec possibilité d'ajouter un rôle
Possibilité de retirer une ou plusieurs personnes.

Choix d'affichage

Possibilité d'affichage (en cochant la ou les cases) :  
Avec des liens cliquables vers les pages des personnes
Avec un portrait (image) de chaque personne
Par ordre alphabétique

Exemples

Vous trouverez différents types d'exemples de mise en page de listes de personnes sur Example.osuny.org
