Comment ajouter des listes de personnes ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 47 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Le bloc de liste personne va vous permettre de mettre en avant une liste de personnes, avec photo, fonction et lien vers la page de la personne.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "c85bafdf-f020-4464-89cb-63c979561ef8" file: "c85bafdf-f020-4464-89cb-63c979561ef8" alt: >- credit: >- - kind: block template: title title: >- Cliquez sur le bloc personnes slug: >- cliquez-sur-le-bloc-personnes ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Cliquez sur le bloc Personnes pour l'ajouter.

notes: >- image: id: "80bc9dd2-a5ef-4c9a-857e-621d4b9f4f38" file: "80bc9dd2-a5ef-4c9a-857e-621d4b9f4f38" alt: >- credit: >- - kind: block template: title title: >- Choisissez les options d'affichage slug: >- choisissez-les-options-d-affichage ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez choisir les options d'affichage de la liste de pages affichée : avec des liens cliquables vers les pages des organisations, avec un portrait de chaque personne et/ou par ordre alphabétique.

notes: >- image: id: "a37e4c64-21e0-4267-832b-bc34f3820a78" file: "a37e4c64-21e0-4267-832b-bc34f3820a78" alt: >- credit: >- - kind: block template: title title: >- Ajouter une personne slug: >- ajouter-une-personne ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "b94e1954-f8a0-4c98-ba09-23de2eb439b8" file: "b94e1954-f8a0-4c98-ba09-23de2eb439b8" alt: >- credit: >- text: >-

Vous pouvez ajouter les personnes que vous souhaitez mettre en avant.

- id: "24460a2f-f9db-4b81-ba9c-6664da97b4a5" file: "24460a2f-f9db-4b81-ba9c-6664da97b4a5" alt: >- credit: >- text: >-

Vous pouvez choisir parmi une liste de personnes enregistrées depuis votre tableau de bord.

- id: "73485026-a1fd-4315-bc0f-b25be6eb67f1" file: "73485026-a1fd-4315-bc0f-b25be6eb67f1" alt: >- credit: >- text: >-

Vous pouvez en sélectionner autant que vous le souhaitez.
