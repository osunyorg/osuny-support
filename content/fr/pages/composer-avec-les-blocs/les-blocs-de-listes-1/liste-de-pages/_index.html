Le bloc Liste de pages va vous permettre de mettre en avant une sélection de pages, avec plusieurs options d'affichage.

notes: >- alt: >- credit: >- - kind: heading title: >- Cliquez sur Ajouter un bloc slug: "cliquez-sur-ajouter-un-bloc" position: 0 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "945531bf-e140-4541-aa0c-d7428de5c5de" file: "945531bf-e140-4541-aa0c-d7428de5c5de" alt: >- credit: >- - kind: heading title: >- Cliquez sur le bloc Pages slug: "cliquez-sur-le-bloc-pages" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Cliquez sur le bloc pages pour l'ajouter.

notes: >- image: id: "45e0c368-04bc-4519-a823-b1a74aaba7ce" file: "45e0c368-04bc-4519-a823-b1a74aaba7ce" alt: >- credit: >- - kind: heading title: >- Choisissez la mise en page slug: "choisissez-la-mise-en-page" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez choisir la mise en page du bloc : une mise en page en grille, en liste, en cartes ou en alternance.

notes: >- image: id: "6b5d0d99-beb3-4efb-b57a-ab94f905bd48" file: "6b5d0d99-beb3-4efb-b57a-ab94f905bd48" alt: >- credit: >- - kind: heading title: >- Choisissez les options d'affichage slug: "choisissez-les-options-d-affichage" position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez choisir les options d'affichage de la liste de pages affichée : afficher une courte description de la page principale, afficher des courtes descriptions des pages sélectionnées, ou encore afficher les images des pages.
