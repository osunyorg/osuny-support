Le bloc de liste Agenda va vous permettre de mettre en avant une liste d'évènements à venir.

notes: >- alt: >- credit: >- - kind: heading title: >- Cliquez sur Ajouter un bloc position: 0 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "5333378d-13f1-4660-816a-e302fe37e37e" file: "5333378d-13f1-4660-816a-e302fe37e37e" alt: >- credit: >- - kind: heading title: >- Cliquez sur le bloc Agenda position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Cliquez sur le bloc Agenda pour l'ajouter.

notes: >- image: id: "87b727e1-4c24-4954-add9-d85ccccc0fe3" file: "87b727e1-4c24-4954-add9-d85ccccc0fe3" alt: >- credit: >- - kind: heading title: >- Choix de mise en page position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Choisissez une mise en page en grille, en liste ou en grand.

notes: >- image: id: "7d49421e-5b64-4b8d-846f-4f8e7bfdc634" file: "7d49421e-5b64-4b8d-846f-4f8e7bfdc634" alt: >- credit: >- - kind: heading title: >- Choisissez les paramètres d'affichage position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez choisir les paramètres d'affichage de la liste d'événements affichés  : afficher le résumé des événements, afficher la catégorie des événements, afficher le statut des événements (en cours, à venir...)
