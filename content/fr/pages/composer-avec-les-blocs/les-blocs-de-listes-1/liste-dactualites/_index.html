Comment ajouter des listes d'actualités ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 42 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Le bloc de listes actualités va vous permettre de mettre en avant une liste d'actualités.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: base: 2 self: 2 top: active: true title: value: >- Cliquez sur Ajouter un bloc heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "128bf68e-fa7d-400a-b31a-72982fed80b1" file: "128bf68e-fa7d-400a-b31a-72982fed80b1" alt: >- credit: >- - kind: block template: title title: >- Cliquez sur le bloc Actualités slug: >- cliquez-sur-le-bloc-actualites ranks: base: 2 self: 2 top: active: true title: value: >- Cliquez sur le bloc Actualités heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Cliquez sur le bloc Actualités pour l'ajouter.

notes: >- image: id: "cdbcea85-890e-4f50-8758-6d6e8c274758" file: "cdbcea85-890e-4f50-8758-6d6e8c274758" alt: >- credit: >- - kind: block template: title title: >- Choisissez quelles actualités afficher slug: >- choisissez-quelles-actualites-afficher ranks: base: 2 self: 2 top: active: true title: value: >- Choisissez quelles actualités afficher heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Vous pouvez choisir les actualités à afficher : toutes les actualités, une catégorie spécifique ou une sélection d'actualités. Vous pouvez également définir un nombre maximum d'actualités à afficher.

notes: >- image: id: "b13c9398-a931-48dd-8884-37a73d097206" file: "b13c9398-a931-48dd-8884-37a73d097206" alt: >- credit: >- - kind: block template: title title: >- Choisissez la mise en page slug: >- choisissez-la-mise-en-page ranks: base: 2 self: 2 top: active: true title: value: >- Choisissez la mise en page heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Vous pouvez choisir la mise en page du bloc : une mise en page en grille, en liste, en cartes ou en alternance.
