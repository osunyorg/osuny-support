Comment ajouter des définitions ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 70 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Le bloc définitions vous permet d'ajouter une liste de termes et leur définition.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Cliquez sur le bloc définitions pour l'ajouter.

notes: >- image: id: "e440db78-dca7-443b-b3f5-17cd802fb525" file: "e440db78-dca7-443b-b3f5-17cd802fb525" alt: >- credit: >- - kind: block template: title title: >- Remplissez le bloc slug: >- remplissez-le-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Rédigez le contenu en ajoutant un nom et une description.

notes: >- image: id: "31a81bdd-5782-4161-b092-0b238332f6ab" file: "31a81bdd-5782-4161-b092-0b238332f6ab" alt: >- credit: >- - kind: block template: title title: >- Ajouter des définitions slug: >- ajouter-des-definitions ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Cliquez sur "Ajouter une définition" pour ajouter un titre et un texte. Vous pouvez ajouter autant de définitions que vous le souhaitez.

Les blocs sont des fonctionnalités que vous pouvez utiliser pour afficher du contenu sur votre site. Ils vous permettent de personnaliser vos pages à l’aide de différents types de contenus comme le texte, les images, les vidéos, etc.

- title: >- Url description: >-

Une URL (sigle de l’anglais : Uniform Resource Locator, littéralement « localisateur uniforme de ressource »), couramment appelée adresse web, est une chaîne de caractères uniforme qui permet d'identifier une ressource du World Wide Web par son emplacement et de préciser le protocole internet pour la récupérer (par exemple http ou https).

- title: >- Navigation web description: >-

La navigation sur le Web fait référence au processus de navigation dans un réseau de ressources d'information sur le World Wide Web, qui est organisé en hypertexte ou en hypermédia.
