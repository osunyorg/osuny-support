Le bloc définitions vous permet d'ajouter une liste de termes et leur définition.

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Cliquez sur le bloc définitions pour l'ajouter.

Rédigez le contenu en ajoutant un nom et une description.

notes: >- image: id: "31a81bdd-5782-4161-b092-0b238332f6ab" file: "31a81bdd-5782-4161-b092-0b238332f6ab" alt: >- credit: >- - kind: heading title: >- Ajouter des définitions slug: "ajouter-des-definitions" position: 4 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Cliquez sur "Ajouter une définition" pour ajouter un titre et un texte. Vous pouvez ajouter autant de définitions que vous le souhaitez.
