Comment ajouter des liens ?

body: >-


Le bloc liens vous permet d'afficher une liste de liens externes.

Ajout d'un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Ajout du bloc Liens

Cliquez sur le bloc liens pour l'ajouter.

Remplir les informations du bloc

Vous pouvez ajouter un nom et une description à votre bloc.

Au clic sur "Ajouter un lien", vous pouvez ajouter un titre, une description, une url et une image.
