Comment ajouter des contacts ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 52 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Le bloc de contact permet d'ajouter des coordonnées ainsi que des horaires d'ouverture, fournissant toutes les informations nécessaires pour être contacté.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "9f249a13-0a02-4697-b916-8bba8d42ed87" file: "9f249a13-0a02-4697-b916-8bba8d42ed87" alt: >- Ajouter un bloc credit: >- - kind: block template: title title: >- Cliquez sur le bloc Contact slug: >- cliquez-sur-le-bloc-contact ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Cliquez sur le bloc contact pour l'ajouter.

notes: >- image: id: "4e236039-4b07-4b1a-9432-aa91bc9c8bae" file: "4e236039-4b07-4b1a-9432-aa91bc9c8bae" alt: >- Choix du bloc contact credit: >- - kind: block template: title title: >- Remplissez le bloc slug: >- remplissez-le-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Rédigez le contenu en ajoutant un nom, une description, une adresse postale, des compléments d'informations et une url d'un site web.

notes: >- image: id: "4052936c-28be-49bc-b7d0-cfa0bb6a5ef6" file: "4052936c-28be-49bc-b7d0-cfa0bb6a5ef6" alt: >- Remplir le bloc credit: >- - kind: block template: title title: >- Ajoutez un téléphone et/ou un mail slug: >- ajoutez-un-telephone-et-ou-un-mail ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Vous pouvez rajouter plusieurs numéros de téléphone et plusieurs mails.

notes: >- image: id: "0636f52e-84e1-492d-91eb-60d29a0d6c37" file: "0636f52e-84e1-492d-91eb-60d29a0d6c37" alt: >- Ajouter un téléphone et un mail si nécessaire credit: >- - kind: block template: title title: >- Ajoutez un créneau horaire slug: >- ajoutez-un-creneau-horaire ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Vous avez la possibilité, si nécessaire, d'ajouter des créneaux horaires.
