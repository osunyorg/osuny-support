Comment ajouter des fichiers ?

body: >-


Le bloc fichiers vous permet d'ajouter une liste de fichiers téléchargeables.

Ajout d'un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Ajout du bloc Fichiers

Cliquez sur le bloc fichiers pour l'ajouter.

Remplir les informations du bloc

Vous pouvez ajouter un nom et une description à votre fichier.

Au clic sur "Ajouter un fichier", vous pouvez ajouter un titre, un fichier et une image.
