Comment ajouter des fichiers ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 41 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Le bloc fichiers vous permet d'ajouter une liste de fichiers téléchargeables, présentés avec leur poids.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "1fcb43c7-70a8-4d3f-b4f6-a3ccdd44f8ef" file: "1fcb43c7-70a8-4d3f-b4f6-a3ccdd44f8ef" alt: >- credit: >- - kind: block template: title title: >- Cliquez sur le bloc Fichiers slug: >- cliquez-sur-le-bloc-fichiers ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Cliquez sur le bloc fichiers pour l'ajouter.

notes: >- image: id: "3617e8e2-1537-4782-bf0c-45624b645c5a" file: "3617e8e2-1537-4782-bf0c-45624b645c5a" alt: >- credit: >- - kind: block template: title title: >- Remplissez le bloc slug: >- remplissez-le-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Rédigez le contenu en ajoutant un nom et une description.

notes: >- image: id: "a178ac54-e2f4-4961-b485-e0b3f1481215" file: "a178ac54-e2f4-4961-b485-e0b3f1481215" alt: >- credit: >- - kind: block template: title title: >- Ajouter un fichier slug: >- ajouter-un-fichier ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >- notes: >- alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "e5f9c8f3-ba9a-46ab-8482-a02882333bde" file: "e5f9c8f3-ba9a-46ab-8482-a02882333bde" alt: >- credit: >- text: >-

Cliquez sur "Ajouter une fichier" pour ajouter un ou plusieurs fichiers.

- id: "bb7c84b0-693f-4b97-b9e4-77caccd6e7cd" file: "bb7c84b0-693f-4b97-b9e4-77caccd6e7cd" alt: >- credit: >- text: >-

Une zone d'ajout apparait. Vous pouvez indiquer un titre et choisir le fichier à charger.

- id: "fa9841e5-b931-4134-9aaf-c8321fbac417" file: "fa9841e5-b931-4134-9aaf-c8321fbac417" alt: >- credit: >- text: >-

Vous pouvez ajouter autant de fichiers que vous le souhaitez.
