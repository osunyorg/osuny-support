Comment ajouter une galerie d'images ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 65 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Ce bloc va vous permettre d'ajouter une galerie d'images, avec pour chaque image une description accessible, un texte et un crédit.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: base: 2 self: 2 top: active: true title: value: >- Cliquez sur Ajouter un bloc heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "1d977dc7-0971-41ce-a215-720fd7dd0e10" file: "1d977dc7-0971-41ce-a215-720fd7dd0e10" alt: >- credit: >- - kind: block template: title title: >- Cliquez sur le bloc Galerie slug: >- cliquez-sur-le-bloc-galerie ranks: base: 2 self: 2 top: active: true title: value: >- Cliquez sur le bloc Galerie heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Cliquez sur le bloc galerie pour l'ajouter.

notes: >- image: id: "c5092dc9-c3e4-48dc-a449-8b1f0a071ae9" file: "c5092dc9-c3e4-48dc-a449-8b1f0a071ae9" alt: >- credit: >- - kind: block template: title title: >- Remplissez le bloc slug: >- remplissez-le-bloc ranks: base: 2 self: 2 top: active: true title: value: >- Remplissez le bloc heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "17c03202-8180-455c-90ce-64ab15975e53" file: "17c03202-8180-455c-90ce-64ab15975e53" alt: >- credit: >- text: >-

Page de création d'une galerie d'images.

- id: "96a81aef-5594-4214-b0f6-698c9bb44c22" file: "96a81aef-5594-4214-b0f6-698c9bb44c22" alt: >- credit: >- text: >-

En cliquant sur "Ajouter une image", vous pouvez ajouter les images une par une.

- id: "fa1e5495-ade4-4d8d-abb6-89aba4eefd61" file: "fa1e5495-ade4-4d8d-abb6-89aba4eefd61" alt: >- credit: >- text: >-

Le bloc Image apparaît en-dessous pour télécharger une image. Vous pouvez entrer le titre de l'image, le crédit et le texte alternatif.

- id: "ad30c7aa-503d-4d7f-bc24-f8fa202302c5" file: "ad30c7aa-503d-4d7f-bc24-f8fa202302c5" alt: >- credit: >- text: >-

Si vous cliquez sur "Sélectionnez vos fichiers", vous pouvez télécharger plusieurs images en même temps.

- id: "30d66c4f-9adc-4ab5-96ae-b7eb40caad46" file: "30d66c4f-9adc-4ab5-96ae-b7eb40caad46" alt: >- credit: >- text: >-

Les blocs d'images correspondant aux images téléchargées sont maintenant affichés. Vous pouvez entrer le titre de l'image, le crédit ainsi que le texte alternatif.

Vous pouvez choisir la mise en page du bloc : une disposition en grille ou en carrousel.
