Comment ajouter une liste de fonctionnalités ?

body: >- Ce bloc vous permet d'insérer une liste de fonctionnalités avec des images, par exemple des icônes, et une description.





Cliquez sur Ajouter un bloc


Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.





Cliquez sur le bloc Fonctionnalités


Cliquez sur le bloc fonctionnalités pour l'ajouter.





Remplissez le bloc






Page d'ajout d'une ou plusieurs fonctionnalités.




En cliquant sur "Ajouter une fonctionnalité", vous pouvez ajouter une fonctionnalité une par une.




Le bloc de fonctionnalité apparaît en dessous. Vous pouvez saisir le titre, le texte et télécharger un fichier.
