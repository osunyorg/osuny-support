Comment ajouter une liste de fonctionnalités ?

body: >-


Ce bloc vous permet d'insérer une liste de fonctionnalités avec des images, par exemple des icônes, et une description.

Ajout d'un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Ajout du bloc Fonctionnalités

Cliquez sur le bloc fonctionnalités pour l'ajouter.

Remplir les informations du bloc

Vous pouvez insérer un nom et une description à votre bloc. Cliquez sur "Ajouter une fonctionnalité" pour ajouter des fonctionnalités.

Vous pouvez rentrer un titre, un texte et une image pour chaque fonctionnalité.

Si une image est associée, les champs "texte alternatif" et "crédit" apparaissent.
