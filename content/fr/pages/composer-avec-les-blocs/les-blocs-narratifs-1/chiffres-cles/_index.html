Comment ajouter des chiffres clés ?

body: >-


Insérez des chiffres clés avec une jolie animation.

Ajout d'un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Ajout du bloc Chiffres-clés

Cliquez sur le bloc Chiffres-clés pour l'ajouter.

Remplissez le bloc

Vous pouvez remplir les informations souhaitées comme un nom ou encore une description. Pour ajouter un chiffre clé, vous devez cliquer sur le bouton "Ajouter un chiffre clé".

Vous pouvez ici rentrer le numéro, l'unité, la description correspondante et une image. Vous pouvez rajouter autant de chiffres clés que vous le souhaitez.
