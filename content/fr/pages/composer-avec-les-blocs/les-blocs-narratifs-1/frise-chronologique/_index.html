Comment insérer une frise chronologique ?

body: >-


Ce bloc vous permet d'ajouter une liste d'événements avec leur description, présentés en frise.

Cliquez sur Ajouter un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Cliquez sur le bloc Frise chronologique

Cliquez sur le bloc Frise chronologique pour l'ajouter.

Remplissez le bloc

En cliquant sur "Ajouter un événement", vous pouvez ajouter un événement un par un.

Le bloc événement apparaît en dessous. Vous pouvez saisir le titre et le texte de l'événement.

Vous pouvez choisir la mise en page du bloc : une mise en page verticale ou horizontale.
