Comment ajouter un appel à actions ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 42 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Ce bloc vous permet d'inviter les personnes à agir en mettant en avant des textes, un visuel et des boutons d'action.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "51b5bf0d-e87b-4ea0-a818-3f5a40831262" file: "51b5bf0d-e87b-4ea0-a818-3f5a40831262" alt: >- credit: >- - kind: block template: title title: >- Cliquez sur le bloc Appel à actions slug: >- cliquez-sur-le-bloc-appel-a-actions ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Cliquez sur le bloc Appel à actions pour l'ajouter.

notes: >- image: id: "df826b64-9c9d-4b0c-a436-cd603dff9455" file: "df826b64-9c9d-4b0c-a436-cd603dff9455" alt: >- credit: >- - kind: block template: title title: >- Remplissez le bloc slug: >- remplissez-le-bloc ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "4780a927-d81a-4571-bbc1-803174a6cf1f" file: "4780a927-d81a-4571-bbc1-803174a6cf1f" alt: >- credit: >- text: >-

Page de création d'un bloc d'appel à l'action. Vous pouvez ajouter un titre, un texte, une image, un texte alternatif et un crédit.

- id: "2cca1337-2a43-4d75-b8c7-94b99c6a4ebe" file: "2cca1337-2a43-4d75-b8c7-94b99c6a4ebe" alt: >- credit: >- text: >-

En cliquant sur "Ajouter un bouton", vous pouvez ajouter un boutons d'action.

- id: "9957ca53-ae93-40d2-b41d-65ab34c5bf12" file: "9957ca53-ae93-40d2-b41d-65ab34c5bf12" alt: >- credit: >- text: >-

Vous pouvez ajouter jusqu'à 3 boutons d'action.
