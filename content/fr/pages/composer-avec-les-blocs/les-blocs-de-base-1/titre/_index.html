Rajoutez des titres pour structurer les contenus de votre page.

body: >-


Le bloc titre va vous permettre de rajouter des titres pour structurer et ranger votre page.

Ajout d'un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Ajout du bloc Titre

Cliquez sur le bloc titre pour l'ajouter.

Remplir les informations du bloc

Vous pouvez choisir entre deux mises en page du bloc : soit une mise en page classique avec uniquement le titre visible ou une mise en page comprenant le titre replié et tous les blocs qui le suivent cachés avec un bouton pour les afficher.
