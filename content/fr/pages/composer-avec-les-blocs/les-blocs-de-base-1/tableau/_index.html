Comment ajouter un tableau ?

Rajoutez des blocs tableau pour illustrer des informations.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Cliquez sur Ajouter un bloc slug: >- cliquez-sur-ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

notes: >- image: id: "270dfe44-ee42-4ed9-8875-7657cbf20c46" file: "270dfe44-ee42-4ed9-8875-7657cbf20c46" alt: >- credit: >- - kind: block template: title title: >- Cliquez sur le bloc Tableau slug: >- cliquez-sur-le-bloc-tableau ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Cliquez sur le bloc Tableau pour l'ajouter.

notes: >- image: id: "5ffa2385-df5d-493c-839c-47338cc84e4f" file: "5ffa2385-df5d-493c-839c-47338cc84e4f" alt: >- credit: >- - kind: block template: title title: >- Remplissez le bloc slug: >- remplissez-le-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Remplissez le bloc en ajoutant un nom, une description.

notes: >- image: id: "43ca6cd4-3c0c-4f68-8faa-75c16b59d98b" file: "43ca6cd4-3c0c-4f68-8faa-75c16b59d98b" alt: >- credit: >- - kind: block template: title title: >- Ajouter des lignes et des colonnes slug: >- ajouter-des-lignes-et-des-colonnes ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Vous pouvez rajouter autant de lignes et de colonnes que vous le souhaitez en cliquant sur les boutons “+ ligne” et “+ colonne”.
