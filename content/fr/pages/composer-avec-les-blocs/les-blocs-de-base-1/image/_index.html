Comment ajouter une image ?

body: >-


Le bloc image va vous permettre de rajouter une image avec sa description alternative, son texte et son crédit.

Ajout d'un bloc

Sur la page ou l'actualité que vous avez créée, cliquez sur "ajouter un bloc" pour commencer à ajouter des blocs de contenu.

Ajout du bloc Image

Cliquez sur le bloc image pour l'ajouter.

Remplir les informations du bloc

Remplissez le bloc en ajoutant un titre, un texte alternatif si nécessaire, une image et un crédit.
