Copier-coller facilement des blocs existants

body: >-


Dans la gestion des blocs, vous pouvez désormais copier-coller des blocs existants d'une page à une autre.

Fonctionnement

Choisissez un bloc à copier et cliquez sur copier le bloc.

Ensuite, allez sur la page ou l'actualité dans laquelle vous souhaitez copier ce bloc, puis cliquez sur "Coller le bloc".
