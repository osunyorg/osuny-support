Comment créer un menu ?

body: >-

Créer un élément du menu

L'utilisateur peut créer un nouvel élément dans le menu en rentrant un titre et choisir un type pour cet élément : 
Titre intermédiaire
Url
Page spécifique
Une fois l'élément créé, il va le retrouver dans l'arborescence de son menu sur la page d'accueil du menu dans "Menu principal". Pour
que dans le menu, un sous menu apparaisse en redirigeant vers une page
du site, il faut choisir comme type "Page spécifique". Il faut ensuite
choisir la cible (la page de redirection). Un titre par défaut apparait
(celui de la page de redirection), l'utilisateur peur changer le nom du
titre de l'élément créé.

Comment créer un menu ?

header_text: >- header_cta: contents_reading_time: seconds: 31 text: >- 1 minute contents: - kind: block template: title title: >- Créer un élément du menu slug: >- creer-un-element-du-menu ranks: base: 2 self: 2 top: active: true title: value: >- Créer un élément du menu heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

L'utilisateur peut créer un nouvel élément dans le menu en rentrant un titre et choisir un type pour cet élément : 

  • Titre intermédiaire
  • Url
  • Page spécifique

Une fois l'élément créé, il va le retrouver dans l'arborescence de son menu sur la page d'accueil du menu dans "Menu principal".

Pour que dans le menu, un sous menu apparaisse en redirigeant vers une page du site, il faut choisir comme type "Page spécifique". Il faut ensuite choisir la cible (la page de redirection). Un titre par défaut apparait (celui de la page de redirection), l'utilisateur peur changer le nom du titre de l'élément créé.
