Tout ce qu'il faut savoir pour les premiers pas avec Osuny

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 125 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Vous trouverez ici les différentes étapes pour démarrer rapidement avec Osuny.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Demander le lien du back office slug: >- demander-le-lien-du-back-office ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Pour contribuer et avoir un accès à l'url du back-office Osuny, vous devez d'abord demander le lien à la personne responsable du site ou de l'extranet.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer son compte slug: >- creer-son-compte ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Une fois que vous avez reçu l'URL du back office, inscrivez-vous en créant un compte. Après la création du compte, vous recevrez un e-mail vous demandant de confirmer votre adresse e-mail pour finaliser l'inscription. Ensuite, connectez-vous au back-office avec vos identifiants.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Demander un accès pour l'utilisation slug: >- demander-un-acces-pour-l-utilisation ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Une fois que votre inscription est finalisée, nous devons vous donner les droits d'accès au back-office du site ou de l'extranet demandé. Nous devons vous attribuer un rôle : contributeur.trice, auteur.e, responsable de site web ou de formation, ou administrateur.rice.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer des pages slug: >- creer-des-pages ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Une fois que l'accès au back office est effectif, vous pouvez contribuer en ajoutant des pages. Une page vous permet de créer et afficher du contenu sur votre site Web ou extranet. Vous pouvez créer autant de pages que vous le souhaitez. Pour chaque page créée, vous devez indiquer les informations essentielles de la page, choisir son format, puis créer du contenu.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer des actualités slug: >- creer-des-actualites ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Vous pouvez également contribuer en ajoutant des actualités. La section "Actualités" vous permet de créer et d'afficher des contenus de type événementiel ou informatif sur votre site web. Vous pouvez créer autant d'actualités que vous le souhaitez et les classer par catégories si nécessaire. Une fois l'actualité créée, vous pouvez choisir de la positionner sur une page dédiée aux actualités ou de la diffuser à différents endroits du site.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Composer avec les blocs slug: >- composer-avec-les-blocs ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Une fois que les pages ou les actualités ont été créées, vous pouvez les structurer. La composition se fait via un système de mise en page comprenant des titres et des blocs de contenu. Vous pouvez ajouter autant de blocs de contenus que vous souhaitez et en changer l'ordre. Vous pouvez décider de les lier ou non à des titres.

Dans un premier temps, vérifiez si l’-email n'a pas atterri dans vos spams. Si ce n'est pas le cas, vous pouvez cliquer sur le lien correspondant sur la page et renvoyer les instructions de confirmation en réindiquant votre e-mail.
