Création et gestion des extranets de l'Établissement.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 47 text: >- 1 minute contents: - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >- image: id: "573e9ace-32be-45bd-b5cf-56105a6b976c" file: "573e9ace-32be-45bd-b5cf-56105a6b976c" alt: >- credit: >- - kind: block template: title title: >- Créer un Extranet slug: >- creer-un-extranet ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Possibilité de créer un extranet en remplissant les informations de base : 

  • Informations (Nom, Domaine, Type d'extranet (école/formation/laboratoire/revue scientifique), Formation (liste déroulante avec toutes les formations rentrées dans l'université)
  • Logo

L’utilisateur.rice en cliquant sur le nom de l'Extranet arrive sur le site de l’extranet et non pas sur le back office comme sur la partie site Web.

Pour accéder aux contenus il faut passer par la partie Alumni - lien externe.

L'utilisateur peut modifier les informations essentielles de l'extranet sélectionné avec les champs suivants :

  • Informations (Nom, Domaine, Type d'extranet (école/formation/laboratoire/revue scientifique), Formation (liste déroulante avec toutes les formations rentrées dans l'université)
  • Logo

Possibilité de rechercher directement par nom et de filtrer par type : Extranet d’une école, d’une formation,  d’un laboratoire ou d’une revue scientifique. Possibilité de réinitialiser la recherche.

 Possibilité de supprimer un extranet. Un message de confirmation apparait avant la suppression définitive.
