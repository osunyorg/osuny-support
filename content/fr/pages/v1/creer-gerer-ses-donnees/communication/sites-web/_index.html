Possibilité de rechercher directement par nom et de filtrer par type : Site d'école, Site de formation, Site de laboratoire, Site de revue scientifique. Possibilité de réinitialiser la recherche.

Au clic sur le nom du site, l'utilisateur.rice est renvoyé.e vers le back office du site.

Possibilité de modifier les informations essentielles du site web sélectionné, mais pas le contenu. Le contenu est modifiable en cliquant sur le nom du site web en accédant au back office.

  • Informations (Nom, Url, Type de site (liste déroulante), École, Langue)
  • Git (Github, Point d'accès Git, Access token, Repository
  • Analytics (Plausible url)

Possibilité de créer un nouveau site web d'une Université en remplissant les champs suivants : 

  • Informations (Nom, Url, Type de site (liste déroulante), École, Langue)
  • Git (Github, Point d'accès Git, Access token, Repository
  • Analytics (Plausible url)
