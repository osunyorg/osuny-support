Création et gestion des diplômes de l’Établissement.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 54 text: >- 1 minute contents: - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >- image: id: "cf2b722c-d0c1-46a8-839a-9632dbbb47ad" file: "cf2b722c-d0c1-46a8-839a-9632dbbb47ad" alt: >- credit: >- - kind: block template: title title: >- Créer un Diplôme slug: >- creer-un-diplome ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

L'utilisateur.rice peut créer un diplôme avec les champs suivants : 

  • Nom
  • Nom abrégé
  • Crédit ECTS
  • Niveau (liste déroulante)
  • Durée

Au clic sur un diplôme, l’utilisateur.rice arrive sur la fiche du diplôme avec les informations suivantes : 

  • Blocs de contenus : possibilité d'ajouter des blocs de contenus - lien externe comme pour les pages, les actualités, etc.
  • Formations  :  liste des formations proposant le diplôme choisi. Au clic sur une formation, accès à la rubrique Formations - lien externe, l'utilisateur.rice a la possibilité de modifier et/ou supprimer une formation directement ici.
  • Informations : Informations essentielles du diplôme.
  • Bouton Modifier en bas à droite : modification de la fiche du diplôme choisi.
  • Bouton supprimer en bas à gauche : possibilité de supprimer un diplôme. Un message de confirmation apparait avant la suppression définitive.
  • Fichier statique : accès aux informations textuelles présentant le diplôme.
