Création et gestion des personnes de l’établissement : Enseignant.e, Personnel administratif, Alumnus, Titulaire, Chercheur.e ou HDR.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 92 text: >- 2 minutes contents: - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >- image: id: "6fddbdcf-4335-4614-9caa-41bb9ef286e3" file: "6fddbdcf-4335-4614-9caa-41bb9ef286e3" alt: >- Personnes de l'établissement credit: >- - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

L'idée de cette section est de pouvoir rentrer toutes les personnes présentes dans l'établissement. Il suffit de créer une fiche sur une personne et de rentrer les informations associées. Ensuite via le bloc de contenu personnes - lien externe, il sera possible d'intégrer une ou plusieurs personnes dans le site où nous le souhaitons, il n'y aura pas besoin à chaque fois de re-rentrer des données sur la personne choisie.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une Personne slug: >- creer-une-personne ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

L'utilisateur peut créer une fiche pour une personne avec les champs suivants :

  • Informations essentielles : Prénom / Nom de famille / Genre / Anniversaire / Description courte / Biographie
  • Informations : Slug / Utilisateur
  • Coordonnées : Email / Téléphone mobile / Téléphone professionnel / Téléphone personnel / Adresse / Code postal / Ville / Pays
  • Photo de profil : Remplacer le fichier
  • SEO : Méta Description
  • Réseaux Sociaux : Site web / Linkedin(url) / Twitter (nom d'utilisateur)
  • Responsabilités : Enseignant.e / Personnel administratif / Alumnus / Titulaire / Chercheur.se / Peut diriger des recherches (HDR)

Au clic sur le nom de la personne, accès à la fiche de cette personne avec ses informations.

Possibilité de supprimer, de modifier la fiche et de générer un fichier statique.
