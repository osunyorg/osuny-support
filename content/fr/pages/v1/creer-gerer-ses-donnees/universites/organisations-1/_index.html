Création et gestion des organisations liées à l’établissement : entreprise, association ou structure gouvernementale.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 90 text: >- 2 minutes contents: - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >- image: id: "db7b19f7-11d3-4fcc-8361-5b98770a757d" file: "db7b19f7-11d3-4fcc-8361-5b98770a757d" alt: >- Liste des organisations credit: >- - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

L'idée de cette section est de pouvoir rentrer toutes les organisations présentes dans l'établissement. Il suffit de créer une fiche sur une organisation et de rentrer les informations associées. Ensuite via le bloc de contenu organisation - lien externe, il sera possible d'intégrer une ou plusieurs organisations dans le site où nous le souhaitons, il n'y aura pas besoin à chaque fois de re rentrer des données sur l'organisation choisie.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une Organisation slug: >- creer-une-organisation ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

L'utilisateur a la possibilité de créer une fiche pour une organisation avec les champs suivants :

  • Contenu : Nom / Texte
  • Informations : Slug / Active ou non
  • Informations de contact : Adresse / Site web / Code Postal / Téléphone / Ville / Email / Pays
  • Logo : Remplacer le fichier
  • SEO : Meta Description
  • Informations légales : Nom /Type (Entreprise/Association/Structure gouvernementale) /Numéro de SIREN
notes: >- image: id: "94fd8f05-1480-4d36-83c4-814a509ce53e" file: "94fd8f05-1480-4d36-83c4-814a509ce53e" alt: >- Création organisation credit: >- - kind: block template: title title: >- Accès à une fiche Organisation slug: >- acces-a-une-fiche-organisation ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Au clic sur le nom de l'organisation, accès à la fiche de l'organisation sélectionnée avec ses informations.

Possibilité de supprimer et/ou modifier la fiche.

L'utilisateur a la possibilité de modifier la fiche de l'organisation sélectionnée avec les champs suivants :

  • Contenu : Nom / Texte
  • Informations : Slug / Active ou non
  • Informations de contact : Adresse / Site web / Code Postal / Téléphone / Ville / Email / Pays
  • Logo : Remplacer le fichier
  • SEO : Meta Description
  • Informations légales : Nom /Type (Entreprise/Association/Structure gouvernementale) /Numéro de SIREN

L'utilisateur.rice a la possibilité de supprimer une organisation. Un message de confirmation apparait avant la suppression définitive.

L'utilisateur.rice peut importer des organisations. Les données doivent être au format xlsx.
Le formatage des données est indiqué au clic sur le bouton "Importer".
L'utilisateur.rice clique sur "choisir un fichier" pour accéder aux fichiers présents sur son espace de travail.
