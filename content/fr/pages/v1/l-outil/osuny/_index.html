Osuny est un commun numérique libre, sobre et accessible pour l'Enseignement Supérieur et la Recherche. C'est une solution technique spécialement conçue pour les universités, laboratoires de recherches et écoles supérieures permettant de créer des sites Web entièrement personnalisés, les plus sobres, les plus accessibles et les plus sécurisés possibles.

notes: >- alt: >- credit: >- - kind: heading title: >- Back-Office simple position: 1 level: 2 - kind: block template: chapter title: > position: 2 data: layout: no_background text: >-

Ergonomie simple, proche de WordPress et intégrant les logiques spécifiques aux Universités et aux écoles.  

notes: >- alt: >- credit: >- - kind: heading title: >- Du code Open Source position: 2 level: 2 - kind: block template: chapter title: > position: 3 data: layout: no_background text: >-

L'outil d'administration est codé avec Ruby on Rails, Bootstrap, et de nombreuses gems open source (Devise, Simple form...).

Le serveur utilisé est Puma.

Le thème Bootstrap d'administration utilisé est Appstack, qui n'est pas Open Source, mais dont noesya a acheté la licence multisite permettant de l'intégrer dans un projet Open Source.

Nous utilisons Kamifusen, la gem open source que nous avons créée, pour optimiser les images servies par l'application Rails. Le code source est disponible sur GitHub.

notes: >- alt: >- credit: >- - kind: heading title: >- Un haut niveau de sécurité position: 3 level: 2 - kind: block template: chapter title: > position: 4 data: layout: no_background text: >-

Le site est précompilé, c'est-à-dire que c'est un ensemble de fichiers HTML, sans langage serveur ni base de données. Il n'y a pas de vulnérabilité de la couche applicative, parce qu'il n'y a pas de couche applicative : pas de Java, pas de PHP, pas de MySQL, PostgreSQL...

L'outil d'administration est hébergé sur le data center Parisien d'Outscale, certifié ISO 27001. Les couches systèmes sont administrées par Scalingo. L'application est dotée d'authentification multifacteur. Le code est surveillé par Code Climate pour la qualité et par GitHub pour la sécurité.

notes: >- alt: >- credit: >- - kind: heading title: >- Hébergement souverain position: 4 level: 2 - kind: block template: chapter title: > position: 5 data: layout: no_background text: >-

La coopérative noesya propose un hébergement commun, avec un silo applicatif par Université. 

L'outil d'administration est hébergé en France par Scalingo, dans le data center Outscale de Paris.
Toutes les données des comptes à privilège sont hébergées en France.

notes: >- alt: >- credit: >- - kind: heading title: >- Migration automatique depuis WordPress position: 5 level: 2 - kind: block template: chapter title: > position: 6 data: layout: no_background text: >-

Afin de faciliter la gestion du changement, Osuny intègre un système d'import depuis WordPress : articles, pages, catégories et auteurs. Les contenus textuels sont importés et nettoyés automatiquement, les visuels sont extraits et stockés pour réutilisation et optimisation dans le site précompilé.

Gestion des comptes utilisateurs sécurisée avec authentification multi-facteur (MFA) et conforme au Règlement Général sur la Protection des Données (RGPD).
