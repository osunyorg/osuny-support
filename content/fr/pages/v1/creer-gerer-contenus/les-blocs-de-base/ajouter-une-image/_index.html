Comment ajouter une image ?

body: >- Ajouter un bloc




Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"



Choisir le bloc image






Remplir le bloc


L'utilisateur peut ajouter un bloc image : une image avec sa description alternative accessible, son texte et son crédit.L'utilisateur rentre les champs suivants pour ajouter un bloc image : 
Titre
Image : choisir un fichier
Publié ou non
Texte alternatif
Crédit
Texte image








Exemples


Vous trouverez différents types d'exemples de mise en page d'images sur Example.osuny.org





Recadrer une image


L'utilisateur peut modifier le cadrage d'une image importer depuis un ordinateur.
