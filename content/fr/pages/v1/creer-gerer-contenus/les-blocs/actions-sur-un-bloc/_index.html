Une section de document avec un titre, un texte, des notes de bas de page et une image.

- title: >- Bloc Image description: >-

Une image avec sa description alternative accessible, son texte et son crédit.

- title: >- Bloc Vidéo description: >-

Une vidéo intégrée dans la page depuis diverses plateformes, avec la transcription et sans lecture automatique.

- title: >- Bloc Tableau description: >-

Un tableau de données, aussi responsive et accessible que possible.

Des outils plus sophistiqués pour articuler et mettre en scène des contenus.

data: description: >-

Des outils plus sophistiqués pour articuler et mettre en scène des contenus.

elements: - title: >- Bloc Chiffres-clés description: >-

Des chiffres clés avec une jolie animation. Chaque chiffre peut être doté d'une unité et d'un texte.

- title: >- Bloc Fonctionnalités description: >-

Une liste de fonctionnalités avec des images, par exemple des icônes, et une description.

- title: >- Bloc Galerie description: >-

Une galerie d'iamges, avec pour chaque image une description accessible, un texte et un crédit.

- title: >- Bloc Appel à actions description: >-

Un titre, un texte, un visuel et 3 boutons pour inviter les personnes à agir.

- title: >- Bloc Témoignages description: >-

Un ou plusieurs témoignages, avec le texte, l'auteur, sa fonction et sa photo.

- title: >- Frise chronologique description: >-

Une liste d'événements avec leur description, présenté en frise.
