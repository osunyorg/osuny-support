L'utilisateur peut ajouter du code HTML brut, permettant d'intégrer des cartes ou divers composants en iframe. 

L'utilisateur rentre les champs suivants pour ajouter un bloc html :

  • Titre
  • Publié ou non
  • Code du bloc d'intégration HTML
  • Transcription textuelle du contenu
