Comment ajouter une liste d'organisations ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 38 text: >- 1 minute contents: - kind: heading title: >- Ajouter un bloc slug: "ajouter-un-bloc" position: 1 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

L'utilisateur peut ajouter une liste de partenaires, avec leur logo, leur site et leur nom. L'utilisateur rentre les champs suivants pour ajouter un bloc partenaires : 

  • Titre
  • Publié ou non
  • Description
  • Ajouter un ou plusieurs partenaires : choix via une liste déroulante des organisations ou rentrer un partenaire : nom / site web / image : choix d'un fichier
  • Possibilité de retirer un ou plusieurs partenaires

Les organisations doivent être créées pour les retrouver dans la liste déroulante, voir la section "Organisations - lien externe".

Possibilité d'affichage (en cochant la ou les cases) :  

  • Avec des liens cliquables vers les pages des organisations
  • Par ordre alphabétique
