Comment ajouter une liste d'actualités ?

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

L'utilisateur peut ajouter une liste d'actualités mises en avant. L'utilisateur rentre les champs suivants pour ajouter un bloc Actualités : 

  • Titre
  • Publié ou non
  • Quelles actualités afficher : Toutes les actualités / Une catégorie spécifique / Une sélection d'actualités
  • Quantité d'articles : champ libre

L'utilisateur.rice a le choix entre plusieurs mises en page pour l'affichage : 

  • Grille
  • Liste
  • Mise en avant
notes: >- image: id: "825c8b6b-c64f-4607-8317-286f8dcd143f" file: "825c8b6b-c64f-4607-8317-286f8dcd143f" alt: >- credit: >- - kind: block template: title title: >- Choix des actualités à afficher slug: >- choix-des-actualites-a-afficher ranks: base: 2 self: 2 top: active: true title: value: >- Choix des actualités à afficher heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

L'utilisateur.rice peut choisir l'affichage des actualités : 

  • Afficher toutes les actualités et il/elle choisit la quantité d'articles à afficher
  • Une catégorie spécifique à travers un choix parmi une liste déroulante de toutes les actualités
  • Une sélection d'actualités en ajoutant les article souhaités. Il faut cliquer sur ajouter un article et le sélectionner parmi la liste déroulante.
