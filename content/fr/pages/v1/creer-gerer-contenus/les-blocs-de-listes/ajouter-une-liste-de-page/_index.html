Comment ajouter une liste de pages ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 81 text: >- 1 minute contents: - kind: block template: title title: >- Ajouter un bloc slug: >- ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

L'utilisateur peut ajouter une mise en avant d'une sélection de pages, avec plusieurs options d'affichage. L'utilisateur rentre les champs suivants pour ajouter un bloc pages :

  • Titre
  • Publié ou non
  • Choisir la mise en page : Grille / Liste / Cartes
  • Type de liste : choix entre une sélection spécifique de pages / La page principale et ses enfants
  • Options d'affichage : Affichage de la description courte de la page principale / Afficher les descriptions courtes des pages / Afficher les images des pages
  • Page principale : Sélection d'une page via une liste déroulante
  • Sélectionner une ou plusieurs pages : liste déroulante
  • Possibilité de retirer une ou plusieurs pages

L'utilisateur.rice a le choix entre plusieurs mises en page pour l'affichage : 

  • Grille
  • Liste
  • Cartes

L'utilisateur.rice peut choisir entre : 

  • Une sélection spécifique de pages : il faut les rajouter manuellement avec le bouton "Sélectionner une page" et choisir la page en question parmi une liste déroulante comprenant toutes les pages du site crées (dans l'arborescence).
  • La page principale et ses enfants : il faut sélectionner une page principale parmi la liste déroulante présente en-dessous du titre "Page Principale". Il s'agit des pages parents crées. Une fois le choix effectuée, automatiquement seront remontées les pages enfants de la page principale sélectionnée.

L'utilisateur.rice a plusieurs options d'affichage pour le bloc de listes de pages. Il peut en sélectionner une ou plusieurs : 

  • Afficher la description courte de la page principale
  • Afficher les descriptions courtes des pages
  • Afficher les images des pages
