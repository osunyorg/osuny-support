Comment ajouter une liste de formations ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 20 text: >- moins d'une minute contents: - kind: block template: title title: >- Ajouter un bloc slug: >- ajouter-un-bloc ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

image: id: "e89ca143-5173-482d-bde4-2a982f8cd83e" file: "e89ca143-5173-482d-bde4-2a982f8cd83e" alt: >- credit: >- - kind: block template: title title: >- Choisir le bloc Formations slug: >- choisir-le-bloc-formations ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >- image: id: "a5815e38-de74-4b70-a63b-7f8644be5d59" file: "a5815e38-de74-4b70-a63b-7f8644be5d59" alt: >- credit: >- - kind: block template: title title: >- Remplir le bloc slug: >- remplir-le-bloc ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

L'utilisateur peut ajouter une liste de formation en cliquant sur ajouter une formation et ensuite les sélectionner via la liste déroulante.

Les formations doivent au préalable être toutes créées, voir la rubrique formations - lien externe.
