Comment créer une actualité ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 60 text: >- 1 minute contents: - kind: block template: title title: >- Créer une actualité slug: >- creer-une-actualite ranks: base: 2 self: 2 top: active: true title: value: >- Créer une actualité heading: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: base: 3 top: active: false data: text: >-

Pour créer une actualité, l'utilisateur doit cliquer sur le bouton "créer" en bas à droite ou "nouvelle curation" s'il s'agit d'un flux RSS.

image: id: "c509e7f4-f2eb-4147-9f58-2de24ee7d44f" file: "c509e7f4-f2eb-4147-9f58-2de24ee7d44f" alt: >- credit: >- - kind: block template: title title: >- Remplir une actualité slug: >- remplir-une-actualite ranks: base: 2 self: 2 top: active: true title: value: >- Remplir une actualité heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Au clic sur "créer", l'utilisateur arrive sur une page vide pour créer une actualité avec les champs suivants : 

  • Contenu :  Titre (obligatoire) / Chapô
  • Catégorie : Choix d'une ou plusieurs catégories définies auparavant
  • SEO :  Meta Description
  • Informations : Publié ou non / Mis en avant ou non / Date de publication / Auteur.rice / Slug
  • Image :  Choisir un fichier / Texte alternatif / Crédit / Chercher une image (Unsplash ou Pexels)

Il clique sur enregistrer pour valider l'actualité.
Possibilité de choisir de publier la page ou non et donc de la mettre en brouillon.

Une fois l'actualité créée, l'utilisateur peut remplir l'actualité par les blocs de contenus (lien externe) en cliquant sur la fiche de l'actualité, sur son nom.
