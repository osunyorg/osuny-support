Possibilité de catégoriser les actualités.

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 43 text: >- 1 minute contents: - kind: block template: title title: >- Ajouter une catégorie slug: >- ajouter-une-categorie ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Pour créer une catégorie, l'utilisateur doit cliquer sur le bouton "créer" à droite de catégorie.

image: id: "9ccf2cd0-6474-458d-b789-255b211599d9" file: "9ccf2cd0-6474-458d-b789-255b211599d9" alt: >- credit: >- - kind: block template: title title: >- Remplir une catégorie slug: >- remplir-une-categorie ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

L'utilisateur peut gérer les catégories créées pour les actualités. Au clic il accède aux informations génériques de la catégorie.

  • Contenu 
  • Informations : slug /catégories enfants (Cet élément est cliquable et nous arrivons sur la liste des actualités de la catégorie enfant sélectionnée)
  • Modifier 
  • Supprimer 
  • Fichier statique

L'utilisateur peut gérer les catégories créées pour les actualités. Au clic il accède aux informations génériques de la catégorie.

  • Contenu :  Nom
  • Informations : slug / catégorie parente (Cet élément est cliquable et nous arrivons sur la liste des actualités de la catégorie parente sélectionnée)
  • Fichier statique
