Comment ajouter du code HTML ?

body: >-

Ajouter un bloc

Rajoutez du contenu dans vos pages en cliquant sur le bouton "ajouter"

Choisir le bloc intégration HTML
Remplir le bloc

L'utilisateur peut ajouter du code HTML brut, permettant d'intégrer des cartes ou divers composants en iframe.  L'utilisateur rentre les champs suivants pour ajouter un bloc html :
Titre
Publié ou non
Code du bloc d'intégration HTML
Transcription textuelle du contenu

Exemples
