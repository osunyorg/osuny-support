Comment structurer l'offre de formation ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 116 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous pouvez gérer ici les formations de votre établissement. Vous pouvez ajouter, modifier ou supprimer des formations.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Creer une formation slug: >- creer-une-formation ranks: base: 2 self: 2 top: active: true title: value: >- Creer une formation heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "66b9f20a-c755-49de-92c3-e2da2f635ef4" file: "66b9f20a-c755-49de-92c3-e2da2f635ef4" alt: >- credit: >- text: >-

Étape 1 - Pour créer une formation, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "a77fe136-fb1c-428c-a8b6-5473f0cdd032" file: "a77fe136-fb1c-428c-a8b6-5473f0cdd032" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Formations".

- id: "eb983301-bf7f-4870-bd28-9d752ab2d751" file: "eb983301-bf7f-4870-bd28-9d752ab2d751" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des formations de votre établissement. Cliquez sur "créer" pour ajouter une formation.

- id: "e22b9ba2-72d4-4e99-80a9-d1359fab70d4" file: "e22b9ba2-72d4-4e99-80a9-d1359fab70d4" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à une fiche pour saisir les informations de la formation que vous souhaitez ajouter : la présentation de l'établissement, les informations liées à la pédagogie, les résultats, les modalités d'admission, ainsi que la possibilité de rajouter une image, de publier ou non la formation.

- kind: block template: title title: >- Ajouter des enseignant.e.s à une formation slug: >- ajouter-des-enseignant-e-s-a-une-formation ranks: base: 2 self: 2 top: active: true title: value: >- Ajouter des enseignant.e.s à une formation heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "e296f51d-c88c-4275-b719-c70f7f3b56a3" file: "e296f51d-c88c-4275-b719-c70f7f3b56a3" alt: >- credit: >- text: >-

Cas d'un ajout d'enseignant.e. sur une formation nouvellement créée.

- id: "1c66f38b-4897-461c-8fbb-b9e6968a28b4" file: "1c66f38b-4897-461c-8fbb-b9e6968a28b4" alt: >- credit: >- text: >-

Lorsque vous êtes sur la création d'une fiche enseignant, vous avez la possibilité d'ajouter des enseignant.e.s.

- id: "5eeb3459-0a7f-4e7f-bb4a-c84b66796fef" file: "5eeb3459-0a7f-4e7f-bb4a-c84b66796fef" alt: >- credit: >- text: >-

En cliquant sur "Ajouter", vous pouvez ajouter autant de personnes que vous le souhaitez via une liste déroulante. Les fiches des enseignants doivent déjà être créées.

- id: "3f309f09-1618-4532-8847-aea4abf8ac01" file: "3f309f09-1618-4532-8847-aea4abf8ac01" alt: >- credit: >- text: >-

Cas d'un ajout d'enseignant.e. sur une formation déjà créée.

- id: "2723d90f-3c2d-45c9-a35d-6e194cbf01df" file: "2723d90f-3c2d-45c9-a35d-6e194cbf01df" alt: >- credit: >- text: >-

En cliquant sur "Ajouter", vous pouvez ajouter autant de personnes que vous le souhaitez via une liste déroulante. Les fiches des enseignants doivent déjà être créées.

- kind: block template: title title: >- Modifier une formation slug: >- modifier-une-formation ranks: base: 2 self: 2 top: active: true title: value: >- Modifier une formation heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "c4e75f3c-421b-4cf5-a293-ffc08c4f4a8a" file: "c4e75f3c-421b-4cf5-a293-ffc08c4f4a8a" alt: >- credit: >- text: >-

En cliquant sur "Formations", vous avez accès à la liste des formations de l'établissement.

- id: "4de42021-9860-476f-8119-cc89446417ec" file: "4de42021-9860-476f-8119-cc89446417ec" alt: >- credit: >- text: >-

En cliquant sur modifier, vous arrivez sur la fiche de la formation sélectionnée.

- id: "509cf568-a632-436d-840c-5f7f2381c318" file: "509cf568-a632-436d-840c-5f7f2381c318" alt: >- credit: >- text: >-

Vous accédez à la fiche de la formation pour en modifier les informations essentielles et gérer la liste des enseignant.e.s.

- kind: block template: title title: >- Gérer les formations slug: >- gerer-les-formations ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les formations heading: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: base: 3 top: active: false data: text: >-

En cliquant sur le nom de la formation, vous accédez à sa fiche, où vous pouvez gérer les enseignant.e.s, les rôles, prévisualiser la fiche ou la modifier.

image: id: "69d46689-9aef-47f3-af16-2960f4991cc4" file: "69d46689-9aef-47f3-af16-2960f4991cc4" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "a358b2d7-bf22-421c-98d3-ef96281b5925" file: "a358b2d7-bf22-421c-98d3-ef96281b5925" alt: >- credit: >- text: >-

Vous pouvez gérer les enseignant.e.s de la formation en cliquant sur "manager".

- id: "ff681641-52c9-4566-be25-45d9f6f12b54" file: "ff681641-52c9-4566-be25-45d9f6f12b54" alt: >- credit: >- text: >-

Vous pouvez gérer les rôles des personnes de la formation sélectionnée en cliquant sur "gérer les rôles".

- id: "bc36c0f7-cefe-4bc0-9b4a-b412a0c9a2ea" file: "bc36c0f7-cefe-4bc0-9b4a-b412a0c9a2ea" alt: >- credit: >- text: >-

Vous pouvez prévisualiser la formation en cliquant sur "prévisualiser".

- id: "cc6c5974-0787-4d91-9f10-9b5cc39c74f2" file: "cc6c5974-0787-4d91-9f10-9b5cc39c74f2" alt: >- credit: >- text: >-

En cliquant sur "Modifier", vous accédez à la fiche de la formation sélectionnée.
