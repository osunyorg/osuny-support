Comment présenter les différentes écoles de l'établissement ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 123 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous pouvez gérer ici les écoles de votre établissement. Vous pouvez ajouter, modifier ou supprimer des écoles.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une école slug: >- creer-une-ecole ranks: base: 2 self: 2 top: active: true title: value: >- Créer une école heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "7e82b28a-e486-487d-b763-7428c5a7735f" file: "7e82b28a-e486-487d-b763-7428c5a7735f" alt: >- credit: >- text: >-

Étape 1 - Pour créer une école, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "e3372006-768d-47bc-9c7c-3ac14daefc37" file: "e3372006-768d-47bc-9c7c-3ac14daefc37" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Écoles".

- id: "6b1daa58-5e44-4df4-8438-d5a3c1e4fcba" file: "6b1daa58-5e44-4df4-8438-d5a3c1e4fcba" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des écoles de votre établissement. Cliquez sur "créer" pour ajouter une école.

- id: "61ba4bc1-95fe-47f3-9ea6-d051e52ce99e" file: "61ba4bc1-95fe-47f3-9ea6-d051e52ce99e" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à une fiche pour saisir les informations de l'école que vous souhaitez ajouter. Vous pouvez associer les formations que vous avez déjà entrées préalablement en cochant les cases correspondantes.

Étape 1 - Pour modifier une école, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "02725673-8cb6-494d-8462-0dbb71c48e39" file: "02725673-8cb6-494d-8462-0dbb71c48e39" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Écoles".

- id: "ad4f8cbc-cc53-40f9-9952-0554df397470" file: "ad4f8cbc-cc53-40f9-9952-0554df397470" alt: >- credit: >- text: >-

Étape 3 - Cliquez sur "Modifier" pour accéder aux informations de l'école.

- id: "07a91fef-3f36-4cc3-85c4-40684e424bad" file: "07a91fef-3f36-4cc3-85c4-40684e424bad" alt: >- credit: >- text: >-

Étape 4 - Vous accédez à la fiche éditable pour la modifier.

- kind: block template: title title: >- Gérer les écoles slug: >- gerer-les-ecoles ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les écoles heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "f47f7602-eb27-4af8-9970-566ea8a23e34" file: "f47f7602-eb27-4af8-9970-566ea8a23e34" alt: >- credit: >- text: >-

Sur la page avec la liste des écoles de votre établissement. Cliquez sur "modifier" pour accéder à l'école sélectionné.

- id: "fc5fd960-dd2b-440e-92a1-e2f7a2a6ddd6" file: "fc5fd960-dd2b-440e-92a1-e2f7a2a6ddd6" alt: >- credit: >- text: >-

Au clic sur "Gérer l'équipe", vous avez la possibilité de gérer les membres de l'équipe

- id: "7599e39a-5c8f-4069-8fb2-1dc2b053070f" file: "7599e39a-5c8f-4069-8fb2-1dc2b053070f" alt: >- credit: >- text: >-

Au clic sur "Modifier", vous accédez à la fiche éditable pour la modifier.

- id: "8fa303e2-2dd7-4341-904b-b5dd2db718d0" file: "8fa303e2-2dd7-4341-904b-b5dd2db718d0" alt: >- credit: >- text: >-

Vous pouvez accéder à la liste des formations dispensées par l'école et choisir une formation en cliquant dessus.

- id: "99169c49-dc53-4da8-b92e-cc94e3193dcd" file: "99169c49-dc53-4da8-b92e-cc94e3193dcd" alt: >- credit: >- text: >-

Vous pouvez accéder à la liste des sites web associés en cliquant sur le site choisi.

- kind: block template: title title: >- Gérer une équipe slug: >- gerer-une-equipe ranks: base: 2 self: 2 top: active: true title: value: >- Gérer une équipe heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "138f758b-d421-4200-8998-9a7933b06116" file: "138f758b-d421-4200-8998-9a7933b06116" alt: >- credit: >- text: >-

En cliquant sur "Écoles", vous avez accès à la liste des écoles de l'établissement.

- id: "960e71e4-58c1-496a-b89f-8b7409b3f01c" file: "960e71e4-58c1-496a-b89f-8b7409b3f01c" alt: >- credit: >- text: >-

En cliquant sur le nom de l'école, vous arrivez sur les informations de l'école sélectionnée.

- id: "19b9681a-91ca-41e7-88b0-80e9cb73a1aa" file: "19b9681a-91ca-41e7-88b0-80e9cb73a1aa" alt: >- credit: >- text: >-

Au clic sur "Gérez l'équipe", vous accédez à la liste des membres de l'équipe classée par rôle au sen de l'établissement.

- id: "8d0a9827-4e6d-4f40-bc8f-6ade35e91175" file: "8d0a9827-4e6d-4f40-bc8f-6ade35e91175" alt: >- credit: >- text: >-

En cliquant sur "Modifier" à côté d'un rôle, vous accédez à la liste des personnes ayant ce rôle. Vous pouvez alors modifier la description du rôle et gérer les personnes, en les supprimant ou en en ajoutant.

- id: "38578782-fe8a-412f-9b0b-6ed3547daf29" file: "38578782-fe8a-412f-9b0b-6ed3547daf29" alt: >- credit: >- text: >-

Lorsque vous cliquez sur "Ajouter", vous pouvez sélectionner une personne dans une liste déroulante de personnes déjà saisies via le tableau de bord.

- id: "8609c918-9487-4430-a13e-4d352f89701b" file: "8609c918-9487-4430-a13e-4d352f89701b" alt: >- credit: >- text: >-

En cliquant sur le nom du rôle, vous accédez à la liste des personnes ayant ce rôle. Vous pouvez retirer des personnes en cliquant sur "retirer".

- id: "6877d494-21eb-48e9-a8f6-1075de2f025b" file: "6877d494-21eb-48e9-a8f6-1075de2f025b" alt: >- credit: >- text: >-

En cliquant sur "Modifier", vous accédez à la gestion des personnes liées au rôle choisi.
