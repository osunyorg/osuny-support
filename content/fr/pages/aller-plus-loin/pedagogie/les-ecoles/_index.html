Comment présenter les différentes écoles de l'établissement ?

body: >-


Vous pouvez gérer ici les écoles de votre établissement. Vous pouvez ajouter, modifier ou supprimer des écoles.

Créer une école

Pour créer une école, commencez par cliquer sur la rubrique "Éducation" dans le tableau de bord. 

Ensuite, cliquez sur la rubrique "Écoles".

Vous avez accès à la liste des écoles déjà créées. Cliquez sur le bouton "Créer" pour créer votre école. 

Vous avez accès à une fiche pour saisir les informations de l'école que vous souhaitez ajouter. Vous pouvez associer les formations que vous avez déjà entrées préalablement en cochant les cases correspondantes.

Modifier une école

Pour modifier une école, commencez par cliquer sur la rubrique "Écoles" dans le tableau de bord.

Cliquez sur l'école pour accéder à sa fiche.

Vous accédez à la fiche éditable pour la modifier.

Vous pouvez modifier les informations et notamment les formations associées à cette école.

Gérer une équipe

En cliquant sur "Écoles", vous avez accès à la liste des écoles de l'établissement.

En cliquant sur le nom de l'école, vous arrivez sur les informations de l'école sélectionnée. Cliquez sur "Gérez l'équipe", pour accéder à la liste des membres de l'équipe.

Vous avez accès à la liste des membres de l'équipe classée par rôle au sein de l'établissement.

En cliquant sur "Modifier" à côté d'un rôle, vous accédez à la liste des personnes ayant ce rôle. Vous pouvez alors modifier la description du rôle et gérer les personnes, en les supprimant ou en en ajoutant.

Pour ajouter une personne dans votre équipe, cliquez sur le bouton "Ajouter".

Vous pouvez sélectionner une personne dans une liste déroulante de personnes déjà saisies via le tableau de bord et lui attribuer un rôle. Cette personne sera automatiquement rajoutée à l'équipe de l'école.
