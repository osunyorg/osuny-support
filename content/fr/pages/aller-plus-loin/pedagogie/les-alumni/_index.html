Présentation et gestion des anciens étudiant.e.s ayant étudié dans les différentes formations.

Étape 1 - Pour créer des alumni, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "25b981f4-fe4d-4750-96b7-f4df52972d86" file: "25b981f4-fe4d-4750-96b7-f4df52972d86" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Alumni".

- id: "eb32e005-4819-4ae4-8257-46a9e785f826" file: "eb32e005-4819-4ae4-8257-46a9e785f826" alt: >- credit: >- text: >-

Étape 3 - Cliquez sur "Importer une promotion" pour rajouter des élèves.

- id: "ce632da5-c556-4332-bd12-759d101a29a7" file: "ce632da5-c556-4332-bd12-759d101a29a7" alt: >- credit: >- text: >-

Étape 4 - Vous disposez de toutes les informations nécessaires pour préparer votre fichier en vue d'une importation parfaite.

Étape 1 - Une fois que vos anciens élèves sont créés, vous avez la possibilité de modifier un élève en cliquant sur son nom.

- id: "f35b9369-1af1-4f00-a282-f12a52c50b2a" file: "f35b9369-1af1-4f00-a282-f12a52c50b2a" alt: >- credit: >- text: >-

Étape 2 - Vous accédez à la fiche de l'alumnus sélectionné.

- id: "c2c118e8-e4fd-4d6f-987b-9ce0016b1184" file: "c2c118e8-e4fd-4d6f-987b-9ce0016b1184" alt: >- credit: >- text: >-

Étape 3 - Gérer les promotions - Vous pouvez cliquer sur "Gérer les promotions" pour gérer les promotions associées à l'alumnus.

- id: "2e5ff4d2-ae53-455f-b5b3-096cbadf0e52" file: "2e5ff4d2-ae53-455f-b5b3-096cbadf0e52" alt: >- credit: >- text: >-

Étape 3 - Sous étape 1 - Vous pouvez modifier la promotion de l'alumnus, en rajouter ou en supprimer.

- id: "c373917d-3ef8-4589-abc8-a76c73f4ae0e" file: "c373917d-3ef8-4589-abc8-a76c73f4ae0e" alt: >- credit: >- text: >-

Étape 3 - Sous étape 2 - Vous pouvez également cliquer sur "Modifier" pour modifier les informations de l'alumnus.

- id: "3e9659e2-2976-4f10-bf0a-a9e71476c8b0" file: "3e9659e2-2976-4f10-bf0a-a9e71476c8b0" alt: >- credit: >- text: >-

Étape 4 - Vous accédez à la fiche de la personne en question depuis le tableau de bord de la rubrique "Personnes" et modifiez les informations en conséquence.
