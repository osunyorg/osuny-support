Comment présenter les diplômes ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 79 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous pouvez gérer ici les diplômes de votre établissement. Vous pouvez ajouter, modifier ou supprimer des diplômes.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer un diplôme slug: >- creer-un-diplome ranks: base: 2 self: 2 top: active: true title: value: >- Créer un diplôme heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "ad3e2f0c-ec1e-487b-9db9-b129387ff156" file: "ad3e2f0c-ec1e-487b-9db9-b129387ff156" alt: >- credit: >- text: >-

Étape 1 - Pour créer un diplôme, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "0e0b224f-7830-40a4-967c-e3dcf3212933" file: "0e0b224f-7830-40a4-967c-e3dcf3212933" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Diplômes".

- id: "1959b1e8-502b-4f53-b962-b3cc4add0785" file: "1959b1e8-502b-4f53-b962-b3cc4add0785" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des formations de votre établissement. Cliquez sur "créer" pour ajouter un diplôme.

- id: "e1cba187-bc94-4fbe-8561-f83079ca62e8" file: "e1cba187-bc94-4fbe-8561-f83079ca62e8" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à une fiche pour saisir les informations du diplôme que vous souhaitez ajouter.

Étape 1 - Pour modifier un diplôme, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord

- id: "5277a75e-4aec-4d0b-8ec9-9b799e9dfdfe" file: "5277a75e-4aec-4d0b-8ec9-9b799e9dfdfe" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Diplômes".

- id: "1b703085-d936-4fd9-bb1f-687c54a86d0f" file: "1b703085-d936-4fd9-bb1f-687c54a86d0f" alt: >- credit: >- text: >-

Étape 3 - Cliquez sur "Modifier" pour accéder au diplôme.

- id: "0a761e68-0cdb-4b66-9235-8a84196486e2" file: "0a761e68-0cdb-4b66-9235-8a84196486e2" alt: >- credit: >- text: >-

Étape 4 - Vous accédez à la fiche éditable pour la modifier.

Vous pouvez gérer vos diplômes en les modifiant ou en les supprimant.

notes: >- image: id: "411ccf92-99f6-443c-a92c-f97d1aa386ce" file: "411ccf92-99f6-443c-a92c-f97d1aa386ce" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "9f30404e-567f-44d4-9331-ab99240418ea" file: "9f30404e-567f-44d4-9331-ab99240418ea" alt: >- credit: >- text: >-

Au clic sur le nom d'une formation, vous arrivez sur les informations essentielles du diplôme.

- id: "b6dc94c2-aace-4910-85fd-395e67e66c91" file: "b6dc94c2-aace-4910-85fd-395e67e66c91" alt: >- credit: >- text: >-

Vous avez la possibilité de gérer les formations associées au diplôme, de les modifier ou de les supprimer.

- id: "22250d76-42ec-4bb7-9303-8fd6e65e147e" file: "22250d76-42ec-4bb7-9303-8fd6e65e147e" alt: >- credit: >- text: >-

Au clic sur "Modifier", vous arrivez sur la page éditable de du diplôme.

- id: "db5caf27-51db-4588-a281-3a29c34fb220" file: "db5caf27-51db-4588-a281-3a29c34fb220" alt: >- credit: >- text: >-

Vous pouvez ajouter des blocs de contenu et des titres comme sur des pages classiques ou des actualités. Pour en savoir plus, consultez la rubrique "Composer avec les blocs" sur ce site de support.
