Comment présenter les diplômes ?

body: >-


Vous pouvez gérer ici les diplômes de votre établissement. Vous pouvez ajouter, modifier ou supprimer des diplômes.

Créer un diplôme

Étape 1 - Pour créer un diplôme, commencez par cliquer sur la rubrique "Éducation" dans le tableau de bord.

Ensuite, cliquez sur la rubrique "Diplômes".

Vous accédez à la liste des diplômes de votre établissement. Cliquez sur "créer" pour ajouter un diplôme.

Vous avez accès à une fiche pour saisir les informations du diplôme que vous souhaitez ajouter. Vous pouvez ajouter des logos de certifications reconnues.

Modifier un diplôme

Pour modifier un diplôme, commencez par cliquer sur la rubrique "Éducation" dans le tableau de bord. Ensuite, cliquez sur la rubrique "Diplômes".

Vous arrivez sur la liste des diplômes, cliquez sur un diplôme pour le modifier. 

Cliquez sur "Modifier" pour accéder aux informations du diplôme.

Vous accédez à la fiche éditable pour la modifier.
