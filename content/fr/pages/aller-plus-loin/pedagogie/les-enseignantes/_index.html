Comment gérer l'équipe pédagogique ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 116 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous pouvez gérer ici l'équipe pédagogique de votre établissement. Vous pouvez ajouter, modifier ou supprimer des enseignant.e.s.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Ajouter un.e enseignant.e slug: >- ajouter-un-e-enseignant-e ranks: base: 2 self: 2 top: active: true title: value: >- Ajouter un.e enseignant.e heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour ajouter des enseignant.e.s, vous devez accéder à la section "Personnes" du back office. Cette section est disponible depuis la page d'accueil du back office, dans la section "Université".

notes: >- image: id: "724e3284-cee2-48d7-81e5-41406d831bab" file: "724e3284-cee2-48d7-81e5-41406d831bab" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "a6526ef0-ee6a-4208-9463-2b2f38662fc8" file: "a6526ef0-ee6a-4208-9463-2b2f38662fc8" alt: >- credit: >- text: >-

Étape 1 - Vous avez accès a la liste de toutes les personnes de votre structure.

- id: "47fa7751-c319-415e-a543-e8bd4c718786" file: "47fa7751-c319-415e-a543-e8bd4c718786" alt: >- credit: >- text: >-

Étape 2 - Cliquez sur "Créer" pour ajouter une personne.

- id: "52869004-eac4-4540-a463-4d53685d1246" file: "52869004-eac4-4540-a463-4d53685d1246" alt: >- credit: >- text: >-

Étape 3 - Vous avez accès à une fiche pour saisir toutes les informations de la personne que vous souhaitez ajouter.

- id: "184712ef-db78-4d34-ac09-b984e4470046" file: "184712ef-db78-4d34-ac09-b984e4470046" alt: >- credit: >- text: >-

Étape 4 - Une fois les informations saisies, vous pouvez ajouter une responsabilité à la personne. Sélectionnez "Enseignant.e" pour la voir apparaître dans la liste des enseignants. Une même personne peut avoir plusieurs responsabilités.

- kind: block template: title title: >- Modifier un.e enseignant.e slug: >- modifier-un-e-enseignant-e ranks: base: 2 self: 2 top: active: true title: value: >- Modifier un.e enseignant.e heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "602712f4-2872-4cdb-a20f-f33441d6993a" file: "602712f4-2872-4cdb-a20f-f33441d6993a" alt: >- credit: >- text: >-

Étape 1 - Pour modifier un.e enseignant.e, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "155e8751-584c-42ac-8b9b-f24c3f914018" file: "155e8751-584c-42ac-8b9b-f24c3f914018" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Enseignant.e.s".

- id: "c3bcf2b9-1f91-4d09-9333-f0ca655c540c" file: "c3bcf2b9-1f91-4d09-9333-f0ca655c540c" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des enseignant.e.s.

- id: "f6fb175d-250c-421d-b986-18a70d430daf" file: "f6fb175d-250c-421d-b986-18a70d430daf" alt: >- credit: >- text: >-

Étape 4 - Cliquez sur le nom de l'enseignant.e.

- id: "63bb3512-0989-4e6e-986c-e263ab3e0af4" file: "63bb3512-0989-4e6e-986c-e263ab3e0af4" alt: >- credit: >- text: >-

Étape 5 - Vous accédez à sa fiche détaillée et vous pouvez la modifier.

- id: "3008008f-85f8-4b09-a43f-36984cd195d5" file: "3008008f-85f8-4b09-a43f-36984cd195d5" alt: >- credit: >- text: >-

Étape 6 - Si vous cliquez sur le bouton "Gérer les formations", vous accéder à la gestion des formations de vos enseignant.e.s. Vous avez la possibilité d'ajouter, de modifier ou de supprimer des formations associées.

- kind: block template: title title: >- Gérer les enseignant.e.s slug: >- gerer-les-enseignant-e-s ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les enseignant.e.s heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "541c4615-db5f-480e-b78d-6757746a1295" file: "541c4615-db5f-480e-b78d-6757746a1295" alt: >- credit: >- text: >-

Étape 1 - Pour gérer votre équipe pédagogique, commencez par cliquer sur la rubrique "Enseignement" dans le tableau de bord.

- id: "67e1062d-d0c9-4a33-b89a-366d115ffd92" file: "67e1062d-d0c9-4a33-b89a-366d115ffd92" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Enseignant.e.s".

- id: "41fa0402-a062-4f0b-bae7-f25b0c50a7c6" file: "41fa0402-a062-4f0b-bae7-f25b0c50a7c6" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des enseignant.e.s pour pouvoir gérer votre équipe pédagogique.

- id: "8d63363e-f81a-4517-a36b-7ac692ca20bb" file: "8d63363e-f81a-4517-a36b-7ac692ca20bb" alt: >- credit: >- text: >-

Étape 4 - Sous-étape 1 - Si vous cliquez sur le bouton "Modifier", vous pouvez gérer les formations de vos enseignant.e.s en ajoutant, modifiant ou supprimant les formations associées.

- id: "246ed26c-158b-430c-9a1e-a55219f85bd9" file: "246ed26c-158b-430c-9a1e-a55219f85bd9" alt: >- credit: >- text: >-

Étape 4 - Sous-étape 2 - Si vous cliquez sur le nom de l'enseignant.e, vous accédez à sa fiche détaillée et pouvez la modifier. Pour accéder à la gestion des formations, cliquez sur "Gérer les formations" (voir détail étape 4.1).
