Comment gérer l'équipe pédagogique ?

body: >-


Vous pouvez gérer ici l'équipe pédagogique de votre établissement. Vous pouvez ajouter, modifier ou supprimer des enseignant.e.s.

Ajouter un.e enseignant.e

Pour ajouter des enseignant.e.s, vous devez accéder à la section "Personnes" du back-office. Cette section est disponible depuis la page d'accueil du back-office, dans la section "Répertoire".

Vous avez accès à la liste des personnes de votre structure. Cliquez sur "Créer" pour ajouter une fiche. 

Vous pouvez remplir les informations de la personne.

Vous allez indiquer ici si la personne est enseignant.e ou non pour qu'elle puisse ensuite remonter dans la liste des enseignant.e.s.

Modifier une fiche d'un.e enseignant.e

Pour modifier un.e enseignant.e, commencez par cliquer sur la rubrique "Éducation" dans le tableau de bord.

Ensuite, cliquez sur la rubrique "Enseignant.e.s".

Vous accédez à la liste des enseignant.e.s, cliquez sur la fiche de l'enseignant.e que vous souhaitez gérer.

Vous avez accès uniquement aux missions d'enseignement. Si vous souhaitez modifier la fiche complètement, un lien vous permet d'être rebasculé sur la fiche complète dans la rubrique "Personnes" du répertoire.

Gérer les enseignant.e.s

Pour gérer un.e enseignant.e, commencez par cliquer sur la rubrique "Éducation" dans le tableau de bord. Ensuite, cliquez sur la rubrique "Enseignant.e.s". 

Vous accédez à la liste des enseignant.e.s, cliquez sur la fiche de l'enseignant.e que vous souhaitez gérer.

Vous avez accès uniquement aux missions d'enseignement. Si vous souhaitez gérer ces missions, cliquez sur le bouton "Gérer les formations" en haut à droite. 

Vous pouvez ici rajouter des formations à l'enseignant.e, modifier les formations existantes ainsi que le rôle de l'enseignat.e.
