Comment gérer les thèses ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 83 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Gestion des thèses des doctorant.e.s de l'établissement.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Ajouter une thèse slug: >- ajouter-une-these ranks: base: 2 self: 2 top: active: true title: value: >- Ajouter une thèse heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "7863147e-f6c0-4064-98a9-9f0fd276811b" file: "7863147e-f6c0-4064-98a9-9f0fd276811b" alt: >- credit: >- text: >-

Étape 1 - Pour ajouter une thèse, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "46c8528d-c9ae-418c-a4f2-e69741754e8c" file: "46c8528d-c9ae-418c-a4f2-e69741754e8c" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Thèses".

- id: "df56fd11-c8bc-4973-af64-e881f40fe679" file: "df56fd11-c8bc-4973-af64-e881f40fe679" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des thèses de votre établissement. Cliquez sur "créer" pour ajouter un thèsee.

- id: "d0024412-b0fd-4306-9ed5-dceba97ad41e" file: "d0024412-b0fd-4306-9ed5-dceba97ad41e" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à une fiche pour saisir les informations de la thèse que vous souhaitez ajouter. Vous pouvez indiquer le titre, un résumé, et lier cette thèse à un laboratoire, un.e auteur.e et un.e directeur.rice de thèse à partir de listes déroulantes.

1/ Pour modifier une thèse, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "3056726f-d141-4468-a0f9-bcd940e869d9" file: "3056726f-d141-4468-a0f9-bcd940e869d9" alt: >- credit: >- text: >-

2/ Ensuite, cliquez sur la rubrique "Thèses".

- id: "8bdbae6c-1dad-46f2-967e-77dcd9d25454" file: "8bdbae6c-1dad-46f2-967e-77dcd9d25454" alt: >- credit: >- text: >-

3/ Cliquez sur "Modifier" pour accéder aux informations de la thèse.

- id: "f4a1e3d3-145f-4d94-85cf-ff174a076e82" file: "f4a1e3d3-145f-4d94-85cf-ff174a076e82" alt: >- credit: >- text: >-

4/ Vous accédez à la fiche éditable pour la modifier.

- kind: block template: title title: >- Gérer les thèses slug: >- gerer-les-theses ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les thèses heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "1bd386e1-959d-43b6-bfe9-5839f82b2226" file: "1bd386e1-959d-43b6-bfe9-5839f82b2226" alt: >- credit: >- text: >-

En cliquant sur "Thèses", vous avez accès à la liste des laboratoires de l'établissement.

- id: "b7a322d8-1bb4-46ef-bd37-df1819a470cf" file: "b7a322d8-1bb4-46ef-bd37-df1819a470cf" alt: >- credit: >- text: >-

En cliquant sur "Modifier", vous accédez aux informations du laboratoire.

- id: "e3dd484c-7bea-4409-922f-084a6457e671" file: "e3dd484c-7bea-4409-922f-084a6457e671" alt: >- credit: >- text: >-

Accès à la fiche éditable pour la modifier.

- id: "7c545b19-061f-42f0-a7eb-00d8f0c85c95" file: "7c545b19-061f-42f0-a7eb-00d8f0c85c95" alt: >- credit: >- text: >-

En cliquant sur le nom de la thèse, vous accédez à la fiche de la thèse.

- id: "b3799c2e-d97b-497b-98f2-b2d540b3f782" file: "b3799c2e-d97b-497b-98f2-b2d540b3f782" alt: >- credit: >- text: >-

En cliquant sur "Modifier", vous pouvez accéder à la fiche éditable. En cliquant sur le nom de l'auteur.e, du laboratoire ou du directeur.rice de thèse, vous arrivez sur sa fiche pour consulter les informations essentielles et les modifier si nécessaire.
