Comment organiser les revues scientifiques ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 109 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Gestion de vos revues scientifiques, journaux en accès ouvert (open access). Les revues fonctionnent avec des volumes, lesquels contiennent des papiers.

Étape 1 - Pour créer une revue, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "6d2934dd-3272-4686-9a03-11b64e7e74f4" file: "6d2934dd-3272-4686-9a03-11b64e7e74f4" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Revues scientifiques".

- id: "50f6593b-c1f4-4aef-abfa-292b3b1c6429" file: "50f6593b-c1f4-4aef-abfa-292b3b1c6429" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des revues de votre établissement. Cliquez sur "créer" pour ajouter une revue.

- id: "0b4a2583-8213-43b1-a4ec-b56a24f87b35" file: "0b4a2583-8213-43b1-a4ec-b56a24f87b35" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à une fiche pour saisir les informations de la revue que vous souhaitez ajouter.
