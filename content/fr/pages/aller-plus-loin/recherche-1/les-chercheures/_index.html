Comment gérer l'équipe de chercheur.e.s ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 89 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Gestion de votre équipe de chercheur.e.s, personnes liées à votre université, à une composante, à un laboratoire, etc.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Ajouter un chercheur.e slug: >- ajouter-un-chercheur-e ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Pour ajouter des chercheur.e.s, vous devez accéder à la section "Personnes" du back office. Cette section est disponible depuis la page d'accueil du back office, dans la section "Université".

notes: >- image: id: "0796fdc2-6d02-47a8-8ff3-cc6fea096585" file: "0796fdc2-6d02-47a8-8ff3-cc6fea096585" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "63fbc3d0-86c2-47e6-89f8-aff5c2ffaf29" file: "63fbc3d0-86c2-47e6-89f8-aff5c2ffaf29" alt: >- credit: >- text: >-

Étape 1 - Vous avez accès a la liste de toutes les personnes de votre structure.

- id: "de03c202-4ac7-478f-b6d2-df8344b55cfe" file: "de03c202-4ac7-478f-b6d2-df8344b55cfe" alt: >- credit: >- text: >-

Étape 2 - Cliquez sur "Créer" pour ajouter une personne.

- id: "a9918ac5-af3f-4a3d-8dff-e85816f76bd4" file: "a9918ac5-af3f-4a3d-8dff-e85816f76bd4" alt: >- credit: >- text: >-

Étape 3 - Vous avez accès à une fiche pour saisir toutes les informations de la personne que vous souhaitez ajouter.

- id: "4acb4591-73dc-4e73-87ce-91a59062fa3f" file: "4acb4591-73dc-4e73-87ce-91a59062fa3f" alt: >- credit: >- text: >-

Étape 4 - Une fois les informations saisies, vous pouvez ajouter une responsabilité à la personne en sélectionnant "Chercheur.e". Une même personne peut avoir plusieurs responsabilités.

- kind: block template: title title: >- Modifier un.e chercheur.e slug: >- modifier-un-e-chercheur-e ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "5be9be7b-659e-488a-bf61-ed862f78f990" file: "5be9be7b-659e-488a-bf61-ed862f78f990" alt: >- credit: >- text: >-

Étape 1 - Pour modifier un.e chercheur.e, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "e051e541-688d-433b-9c16-4be62d914991" file: "e051e541-688d-433b-9c16-4be62d914991" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Chercheur.e".

- id: "327d2e96-c7c8-4e28-aa90-2c5ae0df161f" file: "327d2e96-c7c8-4e28-aa90-2c5ae0df161f" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des chercheur.e.s

- id: "293239e1-f855-4fcf-b275-534a1fe45ce9" file: "293239e1-f855-4fcf-b275-534a1fe45ce9" alt: >- credit: >- text: >-

Étape 4 - Cliquez sur le nom du chercheur.

- id: "f168b412-4bbf-4828-8dc2-183cba308bd3" file: "f168b412-4bbf-4828-8dc2-183cba308bd3" alt: >- credit: >- text: >-

Étape 5 - Vous accédez à sa fiche détaillée et vous pouvez la modifier en cliquant tout en bas de la fiche sur "Modifier".

Étape 1 - Pour accéder à la liste des chercheur.e.s, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "1a36e43c-572a-459f-a4ff-351052a07dfe" file: "1a36e43c-572a-459f-a4ff-351052a07dfe" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Chercheur.e.s".

- id: "7c96f7e9-a978-4527-997e-c35dd116abf8" file: "7c96f7e9-a978-4527-997e-c35dd116abf8" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des chercheur.e.s de votre établissement. Cliquez sur le nom d'une personne pour accéder à sa fiche.

- id: "a52c4e7e-f50f-41ed-96ba-841fe89ba171" file: "a52c4e7e-f50f-41ed-96ba-841fe89ba171" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à la fiche du chercheur sélectionné.
