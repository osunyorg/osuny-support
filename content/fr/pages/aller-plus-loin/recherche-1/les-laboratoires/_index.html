Gestion des structures de recherche hébergeant le travail des chercheur.e.s.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer un laboratoire slug: >- creer-un-laboratoire ranks: base: 2 self: 2 top: active: true title: value: >- Créer un laboratoire heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "d5b4ccdc-2bfc-460d-9fdd-cafdf7133650" file: "d5b4ccdc-2bfc-460d-9fdd-cafdf7133650" alt: >- credit: >- text: >-

Étape 1 - Pour créer un laboratoire, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "6d71bfcd-1be1-4327-8433-f9afd5066220" file: "6d71bfcd-1be1-4327-8433-f9afd5066220" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Laboratoires".

- id: "897e50f4-e252-4c7d-a40c-92e61d8c1ee8" file: "897e50f4-e252-4c7d-a40c-92e61d8c1ee8" alt: >- credit: >- text: >-

Étape 3 - Vous accédez à la liste des laboratoires de votre établissement. Cliquez sur "créer" pour ajouter un laboratoire.

- id: "fbf02dc9-c1e3-44bc-9457-f9ec57f2732b" file: "fbf02dc9-c1e3-44bc-9457-f9ec57f2732b" alt: >- credit: >- text: >-

Étape 4 - Vous avez accès à une fiche pour saisir les informations du laboratoire que vous souhaitez ajouter.

Étape 1 - Pour modifier une école, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

- id: "b644d834-ccc7-482a-b506-18bda95b82cc" file: "b644d834-ccc7-482a-b506-18bda95b82cc" alt: >- credit: >- text: >-

Étape 2 - Ensuite, cliquez sur la rubrique "Laboratoires".

- id: "03ce19eb-8452-4e67-8aed-653e2a4d298e" file: "03ce19eb-8452-4e67-8aed-653e2a4d298e" alt: >- credit: >- text: >-

Étape 3 - Cliquez sur "Modifier" pour accéder aux informations du laboratoire.

- id: "0ef20563-3a8f-4135-9099-717085380b3e" file: "0ef20563-3a8f-4135-9099-717085380b3e" alt: >- credit: >- text: >-

Étape 4 - Vous accédez à la fiche éditable pour la modifier.

- kind: block template: title title: >- Gérer un laboratoire slug: >- gerer-un-laboratoire ranks: base: 2 self: 2 top: active: true title: value: >- Gérer un laboratoire heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "139f8e0d-33df-4bd6-bf39-87a15992634d" file: "139f8e0d-33df-4bd6-bf39-87a15992634d" alt: >- credit: >- text: >-

En cliquant sur "Laboratoires", vous avez accès à la liste des laboratoires de l'établissement.

- id: "cb930a15-8168-45c8-9485-d5e5536f612f" file: "cb930a15-8168-45c8-9485-d5e5536f612f" alt: >- credit: >- text: >-

En cliquant sur "Modifier", vous accédez aux informations du laboratoire.

- id: "1340e7d9-9b27-4a41-8f3e-13830a51a09c" file: "1340e7d9-9b27-4a41-8f3e-13830a51a09c" alt: >- credit: >- text: >-

Accès à la fiche éditable pour la modifier.

- id: "c32fdf29-4a66-4237-b712-834bb0693c30" file: "c32fdf29-4a66-4237-b712-834bb0693c30" alt: >- credit: >- text: >-

En cliquant sur le nom du laboratoire, vous accédez aux différents axes du laboratoire (non obligatoires).

- id: "18dc63f6-e6e6-4f03-9cec-8e2361c5893e" file: "18dc63f6-e6e6-4f03-9cec-8e2361c5893e" alt: >- credit: >- text: >-

Vous avez la possibilité de créer des axes, de les modifier et de les supprimer.

- id: "58bf13cb-8a1c-4ea3-bce4-b4a11c81ba21" file: "58bf13cb-8a1c-4ea3-bce4-b4a11c81ba21" alt: >- credit: >- text: >-

En cliquant sur "créer", vous accédez à une fiche à remplir avec le nom de l'axe et un texte explicatif.
