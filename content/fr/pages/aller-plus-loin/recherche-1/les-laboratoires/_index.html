Comment gérer les différents laboratoires de recherche ?

body: >-


Gestion des structures de recherche hébergeant le travail des chercheur.e.s.

Créer un laboratoire

Pour créer un laboratoire, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

Ensuite, cliquez sur la rubrique "Laboratoires".

Vous accédez à la liste des laboratoires de votre établissement. Cliquez sur "créer" pour ajouter un laboratoire.

Vous avez accès à une fiche pour saisir les informations du laboratoire que vous souhaitez ajouter.

Modifier un laboratoire

Pour modifier un laboratoire, commencez par cliquer sur la rubrique "Recherche" dans le tableau de bord.

Ensuite, cliquez sur la rubrique "Laboratoires".

Cliquez sur le laboratoire choisi pour accéder aux informations.

Vous accédez à la fiche éditable pour la modifier.

Créer des axes

En cliquant sur "Laboratoires", vous avez accès à la liste des laboratoires de l'établissement.

En cliquant sur le nom du laboratoire choisi, vous accédez aux informations du laboratoire. Vous pouvez créer des axes pour votre laboratoire si vous le souhaitez (non obligatoires).

Vous avez la possibilité de créer des axes, de les modifier et de les supprimer.
