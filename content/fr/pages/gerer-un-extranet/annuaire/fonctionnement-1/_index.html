Comment fonctionne votre annuaire ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 33 text: >- 1 minute contents: - kind: block template: chapter title: >- ranks: self: false data: layout: alt_background text: >-

L'annuaire correspond à l'ensemble des membres de votre structure. Il vous permet d'accéder aux fiches des personnes et des organisations qui la composent.

notes: >- alt: >- credit: >- - kind: heading title: >- Les personnes slug: "les-personnes" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

L'extranet vous permet de gérer la liste des personnes qui ont accès à votre extranet. Vous pouvez ajouter une personne à tout moment, modifier les informations d'une personne ou encore supprimer l'accès d'une personne à votre extranet. Vous pouvez trouver ici - lien externe comment ajouter des personnes.

notes: >- alt: >- credit: >- - kind: heading title: >- Les organisations slug: "les-organisations" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

L'extranet vous permet également de gérer la liste des organisations qui ont accès à votre extranet. Vous pouvez ajouter une organisation à tout moment, modifier les informations d'une organisation ou encore supprimer l'accès d'une organisation à votre extranet. Vous pouvez trouver ici - lien externe comment ajouter des organisations.
