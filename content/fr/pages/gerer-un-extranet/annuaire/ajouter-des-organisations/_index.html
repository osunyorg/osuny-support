Comment créer une organisation ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 31 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

La rubrique "Organisations" correspond à la liste des organisations qui ont accès à votre extranet. Voici comment ajouter de nouvelles organisations à votre extranet.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Accès aux organisations dans le tableau de bord slug: >- acces-aux-organisations-dans-le-tableau-de-bord ranks: base: 2 self: 2 top: active: true title: value: >- Accès aux organisations dans le tableau de bord heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour ajouter des organisations, accédez à la section "Organisations" dans le back office. Cette section est accessible depuis la page d'accueil du back office, dans la section "Université".

notes: >- image: id: "eec901fc-45bc-4f12-ac02-4285fd9197c7" file: "eec901fc-45bc-4f12-ac02-4285fd9197c7" alt: >- credit: >- - kind: block template: title title: >- Ajouter une organisation slug: >- ajouter-une-organisation ranks: base: 2 self: 2 top: active: true title: value: >- Ajouter une organisation heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "59017d1b-1756-40fa-8651-a65a97364f93" file: "59017d1b-1756-40fa-8651-a65a97364f93" alt: >- credit: >- text: >-

1/ Vous avez accès à la liste des organisations de votre structure.

- id: "c09e0e7f-063d-400e-9bd8-23f8d512d0c5" file: "c09e0e7f-063d-400e-9bd8-23f8d512d0c5" alt: >- credit: >- text: >-

2/ Cliquez sur "Créer" pour ajouter une organisation.

- id: "3b2cb59d-52e0-4097-8799-9ba502c37855" file: "3b2cb59d-52e0-4097-8799-9ba502c37855" alt: >- credit: >- text: >-

3/ Vous avez accès à une fiche pour saisir toutes les informations de l'organisation que vous souhaitez ajouter.
