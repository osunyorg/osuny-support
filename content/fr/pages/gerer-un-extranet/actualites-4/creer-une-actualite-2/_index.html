Comment créer une actualité ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 131 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

La rubrique "Actualités" vous permet de créer et d'afficher des contenus de type événementiel ou informatif sur votre extranet. Vous pouvez créer autant d'actualités que vous le souhaitez et les classer par catégories si nécessaire.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une actualité slug: >- creer-une-actualite ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Pour créer une nouvelle actualité vous devez cliquez sur l'onglet "Actualité", puis sur sur le bouton "Créer"en haut au-dessus de la liste des actualités.

notes: >- image: id: "fedd1489-65bc-4184-91c2-598e6542eaf4" file: "fedd1489-65bc-4184-91c2-598e6542eaf4" alt: >- credit: >- - kind: block template: title title: >- Indiquer les informations de l'actualité créée slug: >- indiquer-les-informations-de-l-actualite-creee ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une actualité : le titre de votre actualité, un résumé, l'auteur.e et la date de publication

Vous pouvez choisir de publier ou non la page en cochant la case correspondante.

Vous pouvez également décider de lier votre actualité à une catégorie si vous avez déjà créé des catégories, comme expliqué dans la partie "Gestion des catégories" de ce site support.

notes: >- image: id: "845b6311-f71b-4ecb-86fe-0f98700e1ce3" file: "845b6311-f71b-4ecb-86fe-0f98700e1ce3" alt: >- credit: >- - kind: block template: title title: >- Ajouter une image d'illustration slug: >- ajouter-une-image-d-illustration ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "8acbf236-5d4e-4d65-b1f3-11c635f9ad71" file: "8acbf236-5d4e-4d65-b1f3-11c635f9ad71" alt: >- credit: >- text: >-

Ajoutez une image d'illustration à la page en choisissant un fichier depuis votre ordinateur ou votre mobile.

- id: "121ce8ca-5272-4b5f-8eb4-9a7717ec094d" file: "121ce8ca-5272-4b5f-8eb4-9a7717ec094d" alt: >- credit: >- text: >-

Il est possible de recadrer les images téléchargées depuis son ordinateur comme on le souhaite.

- id: "25154d4e-b424-448f-bd43-dcbf8851f301" file: "25154d4e-b424-448f-bd43-dcbf8851f301" alt: >- credit: >- text: >-

L'image est insérée. Vous devez manuellement indiquer le crédit de l'image.

- id: "f067b813-6771-4d68-abc4-705160bddaac" file: "f067b813-6771-4d68-abc4-705160bddaac" alt: >- credit: >- text: >-

Ajoutez une image d'illustration à la page en choisissant une image parmi les bases de données d'images libres de droits d'Unsplash ou de Pexels.

- id: "d4b0152f-113b-49b0-8c02-3cbbf7bfa3e2" file: "d4b0152f-113b-49b0-8c02-3cbbf7bfa3e2" alt: >- credit: >- text: >-

En cliquant sur "Chercher une image", vous accédez aux bases de données d'images libres de droits Unsplash et Pexels. Vous entrez un mot-clé et des images vous sont proposées.

- id: "94c5b2cf-aad8-4cdb-a148-20cc0366aca9" file: "94c5b2cf-aad8-4cdb-a148-20cc0366aca9" alt: >- credit: >- text: >-

Une fois que vous avez sélectionné l'image sur Unsplash ou Pexels, elle sera insérée et le crédit sera automatiquement ajouté.

- kind: block template: title title: >- Ajouter des contenus slug: >- ajouter-des-contenus ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: no_background text: >-

Vous pouvez composer votre page à l'aide d'un système de blocs de contenus. Ces blocs vous permettent d'organiser votre page comme vous le souhaitez. Vous pouvez utiliser des blocs de contenu de base tels que des blocs de texte, d'image, de vidéo ou de son, des blocs narratifs avec des galeries de photos, des témoignages ou encore des appels à l'action, des blocs de listes pour faire des liens entre vos pages, ainsi que des techniques pour insérer une page de contact ou encore du code HTML.

Vous pouvez également rajouter des titres et les associer ou non au bloc de contenu souhaité. Cette gestion des titres et des blocs de contenus va vous permettre d'obtenir une composition de vos pages simple et conviviale.

Retrouver l'explication de tous ces blocs de contenus dans la rubrique "Composer avec les blocs - lien externe".

Une fois les blocs de contenu créés, votre actualité est organisée comme vous pouvez le voir sur la capture d'écran. Vous pouvez déplacer les différents blocs et titres comme vous le souhaitez et réorganiser votre page.
