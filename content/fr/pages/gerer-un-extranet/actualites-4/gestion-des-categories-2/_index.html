Vous avez la possibilité de créer des catégories pour vos actualités, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux actualités d'une catégorie donnée.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une catégorie slug: "creer-une-categorie" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Actualités, puis sur "Créer"en bas à côté de l'espace Catégories.

notes: >- image: id: "86aef20f-f715-4592-bb5d-4276deecb1c3" file: "86aef20f-f715-4592-bb5d-4276deecb1c3" alt: >- credit: >- - kind: heading title: >- Remplir les informations de la catégorie créée slug: "remplir-les-informations-de-la-categorie-creee" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir un identifiant pour l'URL de cette catégorie.

notes: >- image: id: "7cfbe481-c4a6-43d2-90d9-bc12924c77b7" file: "7cfbe481-c4a6-43d2-90d9-bc12924c77b7" alt: >- credit: >- - kind: heading title: >- Relier une actualité à une catégorie slug: "relier-une-actualite-a-une-categorie" position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour associer une actualité à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'actualité et sélectionnez la catégorie souhaitée dans la liste déroulante.

notes: >- image: id: "23e53fce-fb13-424b-ba8a-69b8426a6d45" file: "23e53fce-fb13-424b-ba8a-69b8426a6d45" alt: >- credit: >- - kind: heading title: >- Gérer les catégories slug: "gerer-les-categories" position: 4 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à toutes les actualités de la catégorie sélectionnée.
