Comment créer des catégories ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 67 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous avez la possibilité de créer des catégories pour vos actualités, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux actualités d'une catégorie donnée.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une catégorie slug: >- creer-une-categorie ranks: base: 2 self: 2 top: active: true title: value: >- Créer une catégorie heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Actualités, puis sur "Créer"en bas à côté de l'espace Catégories.

notes: >- image: id: "86aef20f-f715-4592-bb5d-4276deecb1c3" file: "86aef20f-f715-4592-bb5d-4276deecb1c3" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations de la catégorie créée slug: >- remplir-les-informations-de-la-categorie-creee ranks: base: 2 self: 2 top: active: true title: value: >- Remplir les informations de la catégorie créée heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir un identifiant pour l'URL de cette catégorie.

notes: >- image: id: "7cfbe481-c4a6-43d2-90d9-bc12924c77b7" file: "7cfbe481-c4a6-43d2-90d9-bc12924c77b7" alt: >- credit: >- - kind: block template: title title: >- Relier une actualité à une catégorie slug: >- relier-une-actualite-a-une-categorie ranks: base: 2 self: 2 top: active: true title: value: >- Relier une actualité à une catégorie heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour associer une actualité à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'actualité et sélectionnez la catégorie souhaitée dans la liste déroulante.

notes: >- image: id: "23e53fce-fb13-424b-ba8a-69b8426a6d45" file: "23e53fce-fb13-424b-ba8a-69b8426a6d45" alt: >- credit: >- - kind: block template: title title: >- Gérer les catégories slug: >- gerer-les-categories ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les catégories heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à toutes les actualités de la catégorie sélectionnée.

notes: >- image: id: "e8805942-1974-4423-b697-73fef97ff981" file: "e8805942-1974-4423-b697-73fef97ff981" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "5b87dce9-14a8-45f4-931e-694b058c5817" file: "5b87dce9-14a8-45f4-931e-694b058c5817" alt: >- credit: >- text: >-

En cliquant sur le nom de la catégorie, vous accédez aux actualités liées à cette catégorie. Vous avez la possibilité de les modifier directement, ainsi que les informations de la catégorie en cliquant sur "Modifier" en bas à droite de la page.

- id: "ec4cf773-def4-4781-a574-50396bf70a5c" file: "ec4cf773-def4-4781-a574-50396bf70a5c" alt: >- credit: >- text: >-

En cliquant sur le bouton "Modifier", vous accédez aux informations essentielles de la catégorie sélectionnée.
