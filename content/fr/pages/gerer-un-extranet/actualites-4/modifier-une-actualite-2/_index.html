Comment faire des modifications dans une actualité ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 44 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

La modification des informations de votre actualité est différente de la modification des blocs de contenu. Ces actions sont séparées : elles ne se font pas sur la même page et l'accès se fait par des boutons différents.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Modifier les informations d'une actualité slug: >- modifier-les-informations-d-une-actualite ranks: base: 2 self: 2 top: active: true title: value: >- Modifier les informations d'une actualité heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "6238ad68-2a0f-462a-9b53-ad1c69a428bf" file: "6238ad68-2a0f-462a-9b53-ad1c69a428bf" alt: >- credit: >- text: >-

Vous pouvez modifier les informations essentielles de l'actualité depuis la rubrique "Actualité" en cliquant sur "modifier".

- id: "b1e4cc48-48bb-4d56-a2ec-6d302ccc2bfc" file: "b1e4cc48-48bb-4d56-a2ec-6d302ccc2bfc" alt: >- credit: >- text: >-

Vous avez accès aux informations de l'actualité que vous pouvez modifier.

- id: "92390710-f8b4-4bf2-9a76-d114b892f440" file: "92390710-f8b4-4bf2-9a76-d114b892f440" alt: >- credit: >- text: >-

Vous pouvez également accéder aux informations de l'actualité depuis les blocs de contenu en cliquant sur le bouton "Modifier" en bas à droite.

- kind: block template: title title: >- Modifier les blocs de contenu d'une actualité slug: >- modifier-les-blocs-de-contenu-d-une-actualite ranks: base: 2 self: 2 top: active: true title: value: >- Modifier les blocs de contenu d'une actualité heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Vous pouvez modifier les blocs de contenus de l'actualité depuis la rubrique "Actualité" en cliquant sur le nom de la page et non pas sur le bouton "Modifier".
