Comment créer des types de documents ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 59 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous avez la possibilité de créer des types pour vos documents. Cette fonctionnalité permet d'accéder facilement aux documents d'un type donné.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer un type de document slug: >- creer-un-type-de-document ranks: base: 2 self: 2 top: active: true title: value: >- Créer un type de document heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour créer un type, vous devez cliquez sur l'onglet "Documents, puis sur "Créer" à côté de l'espace "Types".

notes: >- image: id: "76700d47-8231-4458-9804-18507b783b8c" file: "76700d47-8231-4458-9804-18507b783b8c" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations du type créé slug: >- remplir-les-informations-du-type-cree ranks: base: 2 self: 2 top: active: true title: value: >- Remplir les informations du type créé heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Vous pouvez entrer ici le nom du type créé et choisir un identifiant pour l'URL de ce type de document.

notes: >- image: id: "9435a077-c54f-46de-ae39-7727ae996c36" file: "9435a077-c54f-46de-ae39-7727ae996c36" alt: >- credit: >- - kind: block template: title title: >- Relier un document à un type de document slug: >- relier-un-document-a-un-type-de-document ranks: base: 2 self: 2 top: active: true title: value: >- Relier un document à un type de document heading: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "e421d370-9330-42bf-8e64-fb6a5f90f80d" file: "e421d370-9330-42bf-8e64-fb6a5f90f80d" alt: >- credit: >- text: >-

Pour associer une document à un type, il faut d'abord créer les types. Ensuite cliquez sur "modifier" sur le document choisi.

- id: "b3e80d8f-53e3-43a2-8829-1706ebcfe957" file: "b3e80d8f-53e3-43a2-8829-1706ebcfe957" alt: >- credit: >- text: >-

Sélectionnez le type choisi dans la liste déroulante pour la lier au document.

- kind: block template: title title: >- Gestion des types slug: >- gestion-des-types ranks: base: 2 self: 2 top: active: true title: value: >- Gestion des types heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Une fois les types créés, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les documents du type sélectionné.

notes: >- image: id: "3802f5e0-c0a4-4266-b733-69b8f5d40277" file: "3802f5e0-c0a4-4266-b733-69b8f5d40277" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "de9d868a-d459-4e33-915c-581cb2307a09" file: "de9d868a-d459-4e33-915c-581cb2307a09" alt: >- credit: >- text: >-

En cliquant sur le bouton "Modifier", vous accédez aux informations essentielles du type sélectionné.

- id: "e02b0fc3-4bdf-4b92-9ea8-dfbb97e3ddec" file: "e02b0fc3-4bdf-4b92-9ea8-dfbb97e3ddec" alt: >- credit: >- text: >-

La fiche du type est éditable.

- id: "01f32ab2-4c7d-40d2-a3eb-9a130f19e74d" file: "01f32ab2-4c7d-40d2-a3eb-9a130f19e74d" alt: >- credit: >- text: >-

En cliquant sur le nom du type, vous accédez aux documents liés à ce type de documents.

- id: "d61ea171-ad09-42eb-b532-7b96c123f05b" file: "d61ea171-ad09-42eb-b532-7b96c123f05b" alt: >- credit: >- text: >-

Vous avez la possibilité de les modifier directement, ainsi que les informations du type. en cliquant sur "Modifier" en bas à droite de la page.
