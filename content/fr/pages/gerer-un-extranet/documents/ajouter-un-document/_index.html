Comment créer un document ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 52 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

La rubrique "Documents" vous permet de créer et d'afficher des documents sur votre extranet. Vous pouvez créer autant de documents que vous le souhaitez et les classer par catégories ou types si nécessaire.

notes: >- alt: >- credit: >- - kind: block template: title title: >- 1/ Créer un document slug: >- 1-creer-un-document ranks: base: 2 self: 2 top: active: true title: value: >- 1/ Créer un document heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour créer un document vous devez cliquez sur l'onglet "Documents", puis sur sur le bouton "Créer"en haut au-dessus de la liste des documents.

notes: >- image: id: "012935b5-d753-4e2d-851e-a6307e865d66" file: "012935b5-d753-4e2d-851e-a6307e865d66" alt: >- credit: >- - kind: block template: title title: >- 2/ Indiquer les informations du document créé slug: >- 2-indiquer-les-informations-du-document-cree ranks: base: 2 self: 2 top: active: true title: value: >- 2/ Indiquer les informations du document créé heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Il s’agit des informations essentielles à remplir lors de la création d'un document : le titre du document et la date de publication. Vous pouvez choisir de publier ou non la page en cochant la case correspondante.

Vous pouvez également décider de lier votre document à un type et une catégorie si vous avez déjà créé des types et des catégories de document, comme expliqué dans la partie "Gestion des types et des catégories" de ce site de support.

notes: >- image: id: "5146fa29-79e9-4f42-8150-3dafb1918304" file: "5146fa29-79e9-4f42-8150-3dafb1918304" alt: >- credit: >- - kind: block template: title title: >- 3/ Document publié slug: >- 3-document-publie ranks: base: 2 self: 2 top: active: true title: value: >- 3/ Document publié heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Une fois le document créé, il apparaît dans la liste des documents. Vous pouvez déplacer les différents documents, les modifier ou les supprimer si vous le souhaitez.
