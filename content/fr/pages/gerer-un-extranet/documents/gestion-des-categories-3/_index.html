Vous avez la possibilité de créer des catégories pour vos documents, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux documents d'une catégorie donnée.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une catégorie slug: "creer-une-categorie" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Documents, puis sur "Créer" à côté de l'espace Catégories.

notes: >- image: id: "ec8048a6-4a14-4fd3-89a6-7632157e8679" file: "ec8048a6-4a14-4fd3-89a6-7632157e8679" alt: >- credit: >- - kind: heading title: >- Remplir les informations de la catégorie créée slug: "remplir-les-informations-de-la-categorie-creee" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir un identifiant pour l'URL de cette catégorie.

Ensuite cliquez sur "modifier" sur le document choisi. - id: "c0fd93ee-9af9-43b1-9052-9119c24eb657" file: "c0fd93ee-9af9-43b1-9052-9119c24eb657" alt: >- credit: >- text: >- Sélectionnez la catégorie choisie dans la liste déroulante pour la lier au document. - kind: heading title: >- Gestion des catégories slug: "gestion-des-categories" position: 4 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les documents de la catégorie sélectionnée.
