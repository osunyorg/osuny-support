Comment créer des catégories de documents ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 65 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Vous avez la possibilité de créer des catégories pour vos documents, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux documents d'une catégorie donnée.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une catégorie slug: >- creer-une-categorie ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Documents, puis sur "Créer" à côté de l'espace Catégories.

notes: >- image: id: "ec8048a6-4a14-4fd3-89a6-7632157e8679" file: "ec8048a6-4a14-4fd3-89a6-7632157e8679" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations de la catégorie créée slug: >- remplir-les-informations-de-la-categorie-creee ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir un identifiant pour l'URL de cette catégorie.

notes: >- image: id: "bece1986-11a2-4e29-ac81-836c2f22dc7a" file: "bece1986-11a2-4e29-ac81-836c2f22dc7a" alt: >- credit: >- - kind: block template: title title: >- Relier un document à une catégorie slug: >- relier-un-document-a-une-categorie ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "e421d370-9330-42bf-8e64-fb6a5f90f80d" file: "e421d370-9330-42bf-8e64-fb6a5f90f80d" alt: >- credit: >- text: >-

Pour associer une document à une catégorie, il faut d'abord créer les catégories. Ensuite cliquez sur "modifier" sur le document choisi.

- id: "c0fd93ee-9af9-43b1-9052-9119c24eb657" file: "c0fd93ee-9af9-43b1-9052-9119c24eb657" alt: >- credit: >- text: >-

Sélectionnez la catégorie choisie dans la liste déroulante pour la lier au document.

- kind: block template: title title: >- Gestion des catégories slug: >- gestion-des-categories ranks: self: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: no_background text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les documents de la catégorie sélectionnée.

notes: >- image: id: "637fc457-6748-4e43-a5c0-5c0d0c846741" file: "637fc457-6748-4e43-a5c0-5c0d0c846741" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "a31bdc7f-3f8a-4875-aba8-0bcf650dcd16" file: "a31bdc7f-3f8a-4875-aba8-0bcf650dcd16" alt: >- credit: >- text: >-

En cliquant sur le nom de la catégorie, vous accédez aux documents liés à cette catégorie. Vous avez la possibilité de les modifier directement, ainsi que les informations de la catégorie en cliquant sur "Modifier" en bas à droite de la page.

- id: "498a2f12-7b9d-44d3-a737-870dc563f403" file: "498a2f12-7b9d-44d3-a737-870dc563f403" alt: >- credit: >- text: >-

En cliquant sur le bouton "Modifier", vous accédez aux informations essentielles de la catégorie sélectionnée.
