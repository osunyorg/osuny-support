Fonction des blocs de contenus

body: >-

Fonctionnement

L'utilisateur peut rajouter du contenu dans les pages par des blocs
de contenus. Ces blocs vont permettre de constituer et structurer une
page. L'utilisateur peut ajouter un bloc de contenu parmi un large
choix de blocs.  Ils sont séparés en 4 groupes, les blocs de base, les
blocs narratifs, les blocs de listes et les blocs techniques. Chaque bloc a son explication, l'utilisateur clique sur "Choisir" pour appliquer son choix. Une fois le bloc choisi, l'utilisateur peut remplir le contenu du bloc.

Les actions

Modifier un bloc : L'utilisateur peut modifier chaque bloc de contenu Dupliquer un bloc : Possibilité sur une page de dupliquer un bloc de contenu Supprimer un bloc : Possibilité de supprimer un bloc. Un message de confirmation apparait avant la suppression définitive.
