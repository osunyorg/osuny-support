Il est maintenant possible de gérer des photothèques et de les utiliser pour illustrer les différents contenus.

Accès à la médiathèque
Ajouter un média
Ajouter une image depuis la médiathèque

Il est possible d'ajouter des images depuis la médiathèque pour illustrer nos images à la une. Pour le moment cette médiathèque ne fonctionne que pour les images à la une uniquement.

Vous pouvez, dans les images à la une, choisir d'utiliser une image de votre médiathèque. Cliquez sur le bouton "Ouvrir depuis la médiathèque".

Vous arrivez sur les images de votre médiathèque, avec l'accès par collections et par catégories.
