Comment modifier un menu existant ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 109 text: >- 2 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Vous pouvez modifier un menu déjà créé en changeant, par exemple, son titre, les liens des pages associées, ou en ajoutant d'autres éléments ou sous-éléments.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Modifier un élément de menu slug: >- modifier-un-element-de-menu ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "295aa20a-b304-44bc-a2c3-17b716cf411e" file: "295aa20a-b304-44bc-a2c3-17b716cf411e" alt: >- credit: >- text: >-

Pour modifier les liens vers les pages ou les URL des éléments du menu, vous devez accéder au menu et sélectionner l'élément à modifier, puis cliquer sur "Modifier".

- id: "feca075e-1d89-442b-98bb-173b62e34590" file: "feca075e-1d89-442b-98bb-173b62e34590" alt: >- credit: >- text: >-

Vous pouvez modifier le titre, le type de lien, ainsi que la cible et l'élément parent.

- kind: block template: title title: >- Voir un élément de menu slug: >- voir-un-element-de-menu ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "b7b5220d-714a-4de2-9a0b-e916ef3865d0" file: "b7b5220d-714a-4de2-9a0b-e916ef3865d0" alt: >- credit: >- text: >-

En cliquant sur "voir", vous accédez au menu et aux éléments du sous-menu s'il y en a.

- id: "da2ce5c1-9471-4aeb-9f22-a5a32a37040e" file: "da2ce5c1-9471-4aeb-9f22-a5a32a37040e" alt: >- credit: >- text: >-

Vous avez la possibilité de créer des sous-éléments si vous le souhaitez.

- kind: block template: title title: >- Rajouter un élément dans le menu de premier niveau slug: >- rajouter-un-element-dans-le-menu-de-premier-niveau ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "e2b5d43e-58f7-4266-98ee-7f5a3028e6fa" file: "e2b5d43e-58f7-4266-98ee-7f5a3028e6fa" alt: >- credit: >- text: >-

1/ Pour modifier un menu de niveau 1, commencez par cliquer sur le menu souhaité, ici le menu principal.

- id: "a9c29ab8-6ee6-4a92-980f-d80383a262fd" file: "a9c29ab8-6ee6-4a92-980f-d80383a262fd" alt: >- credit: >- text: >-

2/ Ensuite, cliquez sur le bouton "Créer" pour ajouter un élément au menu.

- id: "c967358c-4e7e-4646-9d6f-0b40beb5ea4f" file: "c967358c-4e7e-4646-9d6f-0b40beb5ea4f" alt: >- credit: >- text: >-

3/ Pour lier une page existante sur le site, comme la page "L'outil", cliquez sur "Page spécifique" et sélectionnez la page souhaitée dans la liste déroulante correspondant à l’arborescence de votre site.

- id: "500c6fec-37f8-42e3-a560-cc6bc9cc7388" file: "500c6fec-37f8-42e3-a560-cc6bc9cc7388" alt: >- credit: >- text: >-

4/ Le nouvel élément du menu apparaît et vous pouvez modifier l'ordre en déplaçant directement les éléments.

- kind: block template: title title: >- Rajouter un élément dans un sous-menu slug: >- rajouter-un-element-dans-un-sous-menu ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "ee191efe-8406-439c-936f-755cfc18b6e7" file: "ee191efe-8406-439c-936f-755cfc18b6e7" alt: >- credit: >- text: >-

1/ Sélectionnez l'élément du menu auquel vous souhaitez ajouter un sous-élément.

- id: "51041336-a331-4e98-b068-6d63694b9a47" file: "51041336-a331-4e98-b068-6d63694b9a47" alt: >- credit: >- text: >-

2/ Vous arrivez sur la liste des sous éléments.

- id: "53e39b88-2e24-4fc9-9fab-06d6c1f24561" file: "53e39b88-2e24-4fc9-9fab-06d6c1f24561" alt: >- credit: >- text: >-

3/ Pour ajouter un sous-élément, cliquez sur le bouton créer.

- id: "58c9a4d5-4653-4171-bf0d-49bc407651d7" file: "58c9a4d5-4653-4171-bf0d-49bc407651d7" alt: >- credit: >- text: >-

4/ Pour lier une page existante sur le site, comme la page "L'arborescence", cliquez sur "Page spécifique" et sélectionnez la page souhaitée dans la liste déroulante correspondant à l’arborescence de votre site.

- id: "291f8068-2c1a-431a-87c4-7d14b5f657c1" file: "291f8068-2c1a-431a-87c4-7d14b5f657c1" alt: >- credit: >- text: >-

5/ Le nouvel sous-élément du menu apparaît et vous pouvez modifier l'ordre en déplaçant directement les sous-éléments.
