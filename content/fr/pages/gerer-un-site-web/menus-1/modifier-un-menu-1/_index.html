Comment modifier un menu existant ?

body: >-


Vous pouvez modifier un menu déjà créé en changeant, par exemple, son titre, les liens des pages associées, ou en ajoutant d'autres éléments ou sous-éléments.

Modifier un élément de menu

Pour modifier les liens vers les pages ou les URL des éléments du menu, vous devez accéder au menu et sélectionner l'élément à modifier, puis cliquer sur "Modifier".

Vous pouvez modifier le titre, le type de lien, ainsi que la cible et l'élément parent.

Voir un élément de menu

En cliquant sur "voir", vous accédez au menu et aux éléments du sous-menu s'il y en a.

Vous avez la possibilité de créer des sous-éléments si vous le souhaitez.

Rajouter un élément dans le menu de premier niveau

Pour modifier un menu de niveau 1, commencez par cliquer sur le menu souhaité, ici le menu principal.

Ensuite, cliquez sur le bouton "Créer" pour ajouter un élément au menu.

Pour lier une page existante sur le site, comme la page "Equipe", cliquez sur "Page spécifique" et sélectionnez la page souhaitée dans la liste déroulante correspondant à l’arborescence de votre site.

Le nouvel élément du menu apparaît et vous pouvez modifier l'ordre en déplaçant directement les éléments.

Rajouter un élément dans un sous-menu

Sélectionnez l'élément du menu auquel vous souhaitez ajouter un sous-élément.

Vous arrivez sur la liste des sous éléments.

Pour ajouter un sous-élément, cliquez sur le bouton créer.

Pour lier une page existante sur le site, comme la page "L'arborescence", cliquez sur "Page spécifique" et sélectionnez la page souhaitée dans la liste déroulante correspondant à l’arborescence de votre site.

Le nouvel sous-élément du menu apparaît et vous pouvez modifier l'ordre en déplaçant directement les sous-éléments.
