Le menu d'un site web, également appelé menu de navigation ou barre de navigation, est un ensemble de liens qui donne accès aux différentes pages ou sections d'un site. C'est un élément essentiel pour naviguer entre les principaux contenus d'un site internet. 

Cela facilitera la navigation des utilisateurs et leur permettra de trouver rapidement les informations qu'ils cherchent. Cela améliorera également l'expérience utilisateur de votre site, car les visiteurs pourront trouver facilement les pages qu'ils cherchent sans avoir à parcourir tout le site.

Par défaut, vous avez dans l'onglet "Menu" trois types de menus :

  • Primary : correspondant au menu de navigation en haut de votre site
  • Legal : correspondant au menu avec les informations légales en bas du site, dans le footer
  • Social : correspondant au menu avec les liens vers votre présence sur les différents sites sociaux, généralement dans le footer également

Ces trois niveaux vous permettent de structurer vos menus. Pour cela, il est important en amont d'avoir réfléchi à la navigation de votre site et aux informations que vous souhaitez mettre en avant. Il est également nécessaire d'avoir créé vos pages au préalable pour créer des liens. Votre menu de navigation peut être différent de votre arborescence, et les titres doivent être assez courts et compréhensibles.

Si vous n'avez pas en tête le menu de navigation, un menu automatique est créé par défaut pour vous aider au début à trouver les pages créées sur le site. Bien sûr, ce menu est modifiable.

notes: >- alt: >- credit: >- - kind: heading title: >- Construire le menu de premier niveau slug: "construire-le-menu-de-premier-niveau" position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Dans un premier temps, il est nécessaire de créer le premier niveau de navigation, puis d'ajouter des sous-menus si besoin.
