Comment créer un menu et relier des pages ?

body: >- Le menu d'un site web, également appelé menu de navigation ou barre de navigation, est un ensemble de liens qui donne accès aux différentes pages ou sections d'un site. C'est un élément essentiel pour naviguer entre les principaux contenus d'un site internet. Cela facilitera la navigation des utilisateurs et leur permettra de trouver rapidement les informations qu'ils cherchent. Cela améliorera également l'expérience utilisateur de votre site, car les visiteurs pourront trouver facilement les pages qu'ils cherchent sans avoir à parcourir tout le site.





Réfléchir au menu de navigation


Par défaut, vous avez dans l'onglet "Menu" trois types de menus :
Primary : correspondant au menu de navigation en haut de votre site
Legal : correspondant au menu avec les informations légales en bas du site, dans le footer
Social : correspondant au menu avec les liens vers votre présence sur les différents sites sociaux, généralement dans le footer également
Ces trois niveaux vous permettent de structurer vos menus. Pour cela, il est important en amont d'avoir réfléchi à la navigation de votre site et aux informations que vous souhaitez mettre en avant. Il est également nécessaire d'avoir créé vos pages au préalable pour créer des liens. Votre menu de navigation peut être différent de votre arborescence, et les titres doivent être assez courts et compréhensibles.





Menu automatique


Si vous n'avez pas en tête le menu de navigation, un menu automatique est créé par défaut pour vous aider au début à trouver les pages créées sur le site. Bien sûr, ce menu est modifiable.





Construire le menu de premier niveau


Dans un premier temps, il est nécessaire de créer le premier niveau de navigation, puis d'ajouter des sous-menus si besoin.










1/ Exemple d'une page d'un site web sans menu.




2/ Si vous commencez à créer votre barre de navigation de premier niveau dans le menu principal, il vous suffit de cliquer sur le bouton "Créer" pour commencer à concevoir votre menu.




3/ Vous accédez à différents exemples pour créer votre menu et relier des pages. Vous avez le choix entre plusieurs façons de rajouter des pages. Tous les exemples sont détaillés dans la rubrique "exemple de menus".




4/ Pour lier une page existante sur le site, comme la page "Equipe", cliquez sur "Page spécifique" et sélectionnez la page souhaitée dans la liste déroulante correspondant à l’arborescence de votre site. Le titre de la page s'affiche automatiquement. Vous pouvez modifier le titre si vous le souhaitez, surtout si le nom de la page dans la structure de votre site est trop long.




5/ L'élément "Équipe" est maintenant affiché dans votre menu.




6/ Exemple de la même page avec un menu "équipe" créé en haut de la page.


Construire un sous-menu






L'élément "Équipe" est maintenant affiché dans votre menu. Si vous souhaitez créer un sous-menu, il vous suffit de cliquer sur votre menu "Équipe".




Vous voyez apparaître le terme “sous-éléments”, ce qui indique que vous êtes bien dans le menu "Équipe". En cliquant sur "créer", vous pourrez créer des sous-menus de la même manière que des menus de premier niveau.




Pour lier une page existante sur le site, cliquez sur "Page spécifique" et sélectionnez la page souhaitée dans la liste déroulante correspondant à l’arborescence de votre site. Le titre de la page s'affiche automatiquement. Vous pouvez modifier le titre si vous le souhaitez, surtout si le nom de la page dans la structure de votre site est trop long.




L'élément "Équipe éditoriale" est maintenant affiché dans votre menu "Équipe".




Affichage du menu et sous-menu dans le back-office.




L'élément "Équipe" et le sous-élément "Équipe éditoriale" sont maintenant affiché dans le menu de navigation sur le site.
