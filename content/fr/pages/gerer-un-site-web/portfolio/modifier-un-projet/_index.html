Comment modifier un projet ?

body: >-


Vous pouvez modifier les blocs de contenu de votre projet ainsi que les informations liées. Ces actions sont séparées. Pour modifier vos projets, cliquez sur l'onglet "Projets". Ensuite, vous pouvez accéder au projet souhaité en cliquant sur le nom du projet.

Modifier les informations d'un projet

Choisissez le projet que vous souhaitez modifier.

Si vous souhaitez modifier les informations du projet, vous devez cliquer sur le bouton "modifier" en haut à droite.

Vous avez accès aux informations du projet que vous souhaitez modifier.

Modifier les blocs de contenu d'un projet

Choisissez le projet que vous souhaitez modifier.

Vous avez accès aux blocs de contenus que vous pouvez modifier. En cliquant sur "modifier" à côté du bloc de contenu, vous accédez au bloc à modifier.
