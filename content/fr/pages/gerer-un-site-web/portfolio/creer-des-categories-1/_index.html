Vous avez la possibilité de créer des catégories pour vos projets, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux projets d'une catégorie donnée. Un projet peut être classé dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une catégorie slug: "creer-une-categorie" position: 0 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Portfolio", puis sur "Créer" à côté de l'espace Catégories.

image: id: "ffa4cac7-7614-4676-ae5a-69519894d4a8" file: "ffa4cac7-7614-4676-ae5a-69519894d4a8" alt: >- credit: >- - kind: heading title: >- Remplir les informations de la catégorie créée slug: "remplir-les-informations-de-la-categorie-creee" position: 1 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Vous pouvez entrer ici le nom, le résumé de la catégorie et choisir une image.

image: id: "873c341b-c420-4e9e-abba-57ce4d89f930" file: "873c341b-c420-4e9e-abba-57ce4d89f930" alt: >- credit: >- - kind: heading title: >- Relier un projet à une catégorie slug: "relier-un-projet-a-une-categorie" position: 2 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour associer un projet à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations du projet et sélectionnez la catégorie souhaitée.

image: id: "8d562ea6-e96c-406e-bd54-a8e22f3d8ad9" file: "8d562ea6-e96c-406e-bd54-a8e22f3d8ad9" alt: >- credit: >- - kind: heading title: >- Gérer les catégories slug: "gerer-les-categories" position: 3 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les projets de la catégorie sélectionnée.
