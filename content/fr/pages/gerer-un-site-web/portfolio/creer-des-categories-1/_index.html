Comment créer des catégories ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 70 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous avez la possibilité de créer des catégories pour vos projets, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux projets d'une catégorie donnée. Un projet peut être classé dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une catégorie slug: >- creer-une-categorie ranks: base: 2 self: 2 top: active: true title: value: >- Créer une catégorie heading: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: base: 3 top: active: false data: text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Portfolio", puis sur "Créer" à côté de l'espace Catégories.

image: id: "ffa4cac7-7614-4676-ae5a-69519894d4a8" file: "ffa4cac7-7614-4676-ae5a-69519894d4a8" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations de la catégorie créée slug: >- remplir-les-informations-de-la-categorie-creee ranks: base: 2 self: 2 top: active: true title: value: >- Remplir les informations de la catégorie créée heading: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: base: 3 top: active: false data: text: >-

Vous pouvez entrer ici le nom, le résumé de la catégorie et choisir une image.

image: id: "873c341b-c420-4e9e-abba-57ce4d89f930" file: "873c341b-c420-4e9e-abba-57ce4d89f930" alt: >- credit: >- - kind: block template: title title: >- Relier un projet à une catégorie slug: >- relier-un-projet-a-une-categorie ranks: base: 2 self: 2 top: active: true title: value: >- Relier un projet à une catégorie heading: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: base: 3 top: active: false data: text: >-

Pour associer un projet à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations du projet et sélectionnez la catégorie souhaitée.

image: id: "8d562ea6-e96c-406e-bd54-a8e22f3d8ad9" file: "8d562ea6-e96c-406e-bd54-a8e22f3d8ad9" alt: >- credit: >- - kind: block template: title title: >- Gérer les catégories slug: >- gerer-les-categories ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les catégories heading: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: base: 3 top: active: false data: text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les projets de la catégorie sélectionnée.

image: id: "479486de-7b20-4490-98f0-0d5bd2e99a92" file: "479486de-7b20-4490-98f0-0d5bd2e99a92" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "c5f2a690-8a94-488e-872b-0a6684b4b3fe" file: "c5f2a690-8a94-488e-872b-0a6684b4b3fe" alt: >- credit: >- text: >-

En cliquant sur "voir", vous accédez aux blocs de contenu de la catégorie sélectionnée.

- id: "b366c109-faef-4ee4-8209-f0454efe4779" file: "b366c109-faef-4ee4-8209-f0454efe4779" alt: >- credit: >- text: >-

Vous pouvez ajouter ou modifier du contenu de la catégorie choisie. Vous pouvez ajouter ou modifier des titres ou des blocs de contenu.

- id: "5939ccde-a1cd-4fce-974e-16dc8a13af04" file: "5939ccde-a1cd-4fce-974e-16dc8a13af04" alt: >- credit: >- text: >-

En cliquant sur "modifier", vous accédez aux informations essentielles de la catégorie sélectionné.

- id: "f7121e9a-6800-4e5f-97fe-1b84c9ceeabc" file: "f7121e9a-6800-4e5f-97fe-1b84c9ceeabc" alt: >- credit: >- text: >-

Vous pouvez modifier le nom, le résumé ou l'image de la catégorie sélectionné.
