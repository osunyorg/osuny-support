Comment ajouter un portfolio ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 21 text: >- moins d'une minute contents: - kind: block template: chapter title: >- ranks: self: false data: layout: alt_background text: >-

La fonctionnalité Portfolio vous permet d'ajouter des projets et de les classer par année.

notes: >- alt: >- credit: >- - kind: heading title: >- Ajouter un portfolio slug: "ajouter-un-portfolio" position: 1 rank: 2 - kind: block template: gallery title: >- ranks: self: false data: description: >- layout: grid images: - id: "166704f3-9608-4926-a068-bc459d179a92" file: "166704f3-9608-4926-a068-bc459d179a92" alt: >- credit: >- text: >-

Pour ajouter la fonctionnalité portfolio, vous devez cliquer sur "modifier" quand vous êtes sur la page d'accueil de votre site.

- id: "c9fae4dd-a72f-4eb9-a433-11f7170f4bcc" file: "c9fae4dd-a72f-4eb9-a433-11f7170f4bcc" alt: >- credit: >- text: >-

Ensuite vous cochez la case "portfolio" pour l'activer et qu'il apparaisse ensuite dans votre menu sur le côté.
