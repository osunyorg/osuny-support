Comment ajouter un portfolio ?

body: >-


La fonctionnalité Portfolio vous permet d'ajouter des projets et de les classer par année.

Ajouter un portfolio

Pour ajouter la fonctionnalité portfolio, vous devez cliquer sur l'onglet "Réglages" présent sur toutes les pages.

Ensuite, cochez la case "portfolio" pour l'activer.
