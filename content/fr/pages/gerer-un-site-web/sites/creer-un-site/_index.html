Comment créer un site et le modifier ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 179 text: >- 3 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Une fois votre espace créé, vous avez la possibilité de créer un site web. Plusieurs sites web peuvent être créés.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer un site slug: >- creer-un-site ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Pour créer un site web, allez dans la section "Communication" de votre tableau de bord, puis sélectionnez "Site web". Ensuite, cliquez sur le bouton "Créer".

image: id: "97af47f0-3f86-40ea-8b2b-d0e55ed396f9" file: "97af47f0-3f86-40ea-8b2b-d0e55ed396f9" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations éditoriales slug: >- remplir-les-informations-editoriales ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

ll s'agit des informations à remplir lors de la création d'un site web. Commencez par la partie éditoriale qui concerne le contenu du site. Indiquez le nom du site, activez les fonctionnalités souhaitées, choisissez les langues, personnalisez l'apparence de votre site, le fuseau horaire et remplissez la section réseaux sociaux.

image: id: "bdd4d9f2-384f-45c0-a036-05fa56280097" file: "bdd4d9f2-384f-45c0-a036-05fa56280097" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "628b5da1-b716-489b-b33c-e14b70141c61" file: "628b5da1-b716-489b-b33c-e14b70141c61" alt: >- credit: >- text: >-

Activez les fonctionnalités que vous souhaitez pour votre site web : actualités, agenda et portfolio. Les actualités sont activées par défaut. Vous pouvez les garder si vous prévoyez de publier des actualités ou les désactiver. Vous avez aussi la possibilité d'ajouter le module agenda pour créer des événements ou le module portfolio pour présenter des projets.

- id: "5b5b0e02-94ef-4e14-aabf-db3e770f9dc3" file: "5b5b0e02-94ef-4e14-aabf-db3e770f9dc3" alt: >- credit: >- text: >-

Vous avez la possibilité de sélectionner plusieurs langues pour votre site et de définir une langue par défaut.

- id: "69f51067-7e6e-4113-bda3-a8310ba75b74" file: "69f51067-7e6e-4113-bda3-a8310ba75b74" alt: >- credit: >- text: >-

Pour l'apparence de votre site, vous pouvez sélectionner une image par défaut pour illustrer les vignettes de vos pages si vous n'en avez pas. De même, choisissez l'image de partage par défaut de votre site.

- id: "f6ca16d1-df77-447a-ba0e-6e698fe576b1" file: "f6ca16d1-df77-447a-ba0e-6e698fe576b1" alt: >- credit: >- text: >-

La section agenda vous permet de régler le fuseau horaire.

- id: "c3cf7c7d-525f-49a0-b4ec-ebe11ffbdd89" file: "c3cf7c7d-525f-49a0-b4ec-ebe11ffbdd89" alt: >- credit: >- text: >-

Enfin, dans la section Réseaux sociaux, renseignez les URLs des réseaux sociaux de votre site web.

- kind: block template: title title: >- Remplir les informations techniques slug: >- remplir-les-informations-techniques ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Après la partie éditoriale, remplissez la partie technique qui concerne les réglages techniques.  

image: id: "a86e445e-b8b1-471a-8b98-29e7b842f987" file: "a86e445e-b8b1-471a-8b98-29e7b842f987" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "2e06395b-26c7-40d5-bdf1-e5519ec37f5d" file: "2e06395b-26c7-40d5-bdf1-e5519ec37f5d" alt: >- credit: >- text: >-

Ce champ concerne l'URL de votre site web. Il est recommandé de le laisser vide car, lors de la création du site avec l’hébergement Deuxfleurs, il sera rempli automatiquement. Vous pouvez choisir de mettre le site en production ici. Nous vous suggérons de ne pas cocher cette case tant que le site n'est pas hébergé sur un domaine ou un sous-domaine dédié, c'est-à-dire pas sur .osuny.site.

- id: "c93d7ce4-2acb-4cd8-8902-f69ea505467b" file: "c93d7ce4-2acb-4cd8-8902-f69ea505467b" alt: >- credit: >- text: >-

Vous pouvez choisir ici la catégorie de votre site web : site indépendant, site d'école, site de formation, site de laboratoire ou site de revue scientifique.

- id: "17c4a45b-8b4a-49e4-a316-fe164a2ea6b7" file: "17c4a45b-8b4a-49e4-a316-fe164a2ea6b7" alt: >- credit: >- text: >-

Ici, vous avez la possibilité de présenter ou non votre site dans le showcase Osuny et de choisir les mots-clés associés à votre site web. Ce showcase répertorie tous les sites créés avec Osuny.

- id: "fcbdc979-218c-446c-bb11-5d4e2ecd44e5" file: "fcbdc979-218c-446c-bb11-5d4e2ecd44e5" alt: >- credit: >- text: >-

Cette section concerne l'hébergement. Par défaut, un hébergement avec Deuxfleurs est proposé. Ce n'est pas obligatoire vous pouvez avoir votre propre hébergement si vous le souhaitez, merci de nous contacter à ce moment là. Nous vous proposons également la mise à jour automatique du thème Osuny par défaut. Cela signifie que dès qu'il y a des modifications dans le thème Osuny, vous en bénéficierez. Vous pouvez choisir de désactiver cette option. En ce qui concerne le tableau de bord Plausible, vous avez la possibilité d'implémenter des statistiques avec Plausible, une alternative open source à Google Analytics. Il suffit d'entrer un lien de partage généré depuis Plausible.

- id: "3977c6c2-fde3-4129-abfb-a018c46a721d" file: "3977c6c2-fde3-4129-abfb-a018c46a721d" alt: >- credit: >- text: >-

Ces informations techniques concernent le déploiement de votre site. Nous vous recommandons de laisser cette section vide, en particulier si vous créez un site avec un hébergement Deuxfleurs. Nous sommes à votre disposition pour toute question technique.

Le site web créé apparait dans la liste des sites web.

image: id: "38486454-16bd-4f1e-858d-b86a792899b2" file: "38486454-16bd-4f1e-858d-b86a792899b2" alt: >- credit: >- - kind: block template: title title: >- Modifier un site web slug: >- modifier-un-site-web ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Vous avez la possibilité de modifier les informations éditoriales et techniques du site web créé à tout moment. Pour ce faire, cliquez sur "voir" pour le site choisi dans la liste des sites web créés.

image: id: "6c4d9f59-f3fc-40b3-afd2-1ed02059d72b" file: "6c4d9f59-f3fc-40b3-afd2-1ed02059d72b" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >- layout: grid images: - id: "0fc91aa5-698e-4e42-9f6d-f3042f13c5ec" file: "0fc91aa5-698e-4e42-9f6d-f3042f13c5ec" alt: >- credit: >- text: >-

Pour modifier les informations du site, vous devez cliquer sur "modifier" quand vous êtes sur la page d'accueil de l'admin de votre site web.
