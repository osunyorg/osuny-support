Comment gérer les accès ?

body: >-


Vous pouvez gérer les personnes et les organisations de votre structure en les connectant à votre extranet.

Connecter les personnes à l'Extranet

Dans votre espace, cliquez sur "Répertoire" pour y accéder.

Ensuite, vous pouvez connecter toutes les personnes présentes dans votre structure à votre annuaire en cliquant sur "Connecté".

Connecter les organisations à l'extranet

Cliquez sur "Répertoire" pour y accéder.

Ensuite, vous pouvez connecter toutes les organisations présentes dans votre structure à votre annuaire en cliquant sur "Connecté".
