Comment faire des modifications dans une actualité ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 52 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous pouvez modifier les blocs de contenu de votre actualité ainsi que les informations de votre actualité. Ces actions sont séparées. Pour modifier vos actualités, cliquez sur l'onglet "Actualités". Ensuite, vous pouvez accéder à l'actualitée souhaitée en cliquant sur le nom de l'actualité.

Si vous souhaitez modifier les informations de l'actualité, vous devez cliquer sur le bouton "Modifier" en bas à droite.
