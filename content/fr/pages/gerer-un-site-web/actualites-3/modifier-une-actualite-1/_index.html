Comment faire des modifications dans une actualité ?

body: >-


Vous pouvez modifier les blocs de contenu de votre actualité ainsi que les informations de votre actualité. Ces actions sont séparées. Pour modifier vos actualités, cliquez sur l'onglet "Actualités". Ensuite, vous pouvez accéder à l'actualitée souhaitée en cliquant sur le nom de l'actualité.

Modifier les informations d'une actualité

Vous êtes sur l'actualité sélectionnée pour la modifier.

Cliquez sur le bouton modifier

Si vous souhaitez modifier les informations de l'actualité, vous devez cliquer sur le bouton "Modifier" en haut à droite.

Vous avez accès aux informations de l'actualité que vous pouvez modifier.

Modifier les blocs de contenu d'une actualité

Vous êtes sur l'actualité sélectionnée pour la modifier.

Choisissez un bloc de contenu à modifier et cliquez sur le bouton modifier au-dessus du bloc de contenu

Vous avez accès aux blocs de contenu de l'actualité que vous pouvez modifier. En cliquant sur "modifier" à côté du bloc de contenu, vous accédez au bloc à modifier.

Vous pouvez modifier le bloc de contenu sélectionné.
