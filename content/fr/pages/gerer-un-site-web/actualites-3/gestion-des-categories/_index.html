Vous avez la possibilité de créer des catégories pour vos actualités, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux actualités d'une catégorie donnée. Une actualité peut être classée dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une catégorie slug: "creer-une-categorie" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Actualités, puis sur "Créer"en bas à côté de l'espace Catégories.

notes: >- image: id: "8bf2f8d0-b655-43fa-8fa0-3a6b9b68c631" file: "8bf2f8d0-b655-43fa-8fa0-3a6b9b68c631" alt: >- credit: >- - kind: heading title: >- Remplir les informations de la catégorie créée slug: "remplir-les-informations-de-la-categorie-creee" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une nouvelle actualité : le nom de la catégorie, un résumé, et éventuellement une catégorie parente si nécessaire. Vous pouvez également choisir une image pour illustrer votre catégorie.

notes: >- image: id: "3beb3ac7-a3c9-4921-8e09-69c883199618" file: "3beb3ac7-a3c9-4921-8e09-69c883199618" alt: >- credit: >- - kind: heading title: >- Relier une actualité à une ou plusieurs catégories slug: "relier-une-actualite-a-une-ou-plusieurs-categories" position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Pour relier une actualité à une catégorie, il faut tout d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'actualité et cochez la case correspondante à la ou aux catégories souhaitées.

notes: >- image: id: "56baecf5-4f4b-47f3-9fc9-71bf68e79aa7" file: "56baecf5-4f4b-47f3-9fc9-71bf68e79aa7" alt: >- credit: >- - kind: heading title: >- Gérer les catégories slug: "gerer-les-categories" position: 4 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >- notes: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à toutes les actualités de la catégorie sélectionnée.
