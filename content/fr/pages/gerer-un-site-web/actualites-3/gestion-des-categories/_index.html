Comment créer des catégories ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 75 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: base: 2 top: active: false data: layout: alt_background text: >-

Vous avez la possibilité de créer des catégories pour vos actualités, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux actualités d'une catégorie donnée. Une actualité peut être classée dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une catégorie slug: >- creer-une-categorie ranks: base: 2 self: 2 top: active: true title: value: >- Créer une catégorie heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Actualités, puis sur "Créer"en bas à côté de l'espace Catégories.

notes: >- image: id: "8bf2f8d0-b655-43fa-8fa0-3a6b9b68c631" file: "8bf2f8d0-b655-43fa-8fa0-3a6b9b68c631" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations de la catégorie créée slug: >- remplir-les-informations-de-la-categorie-creee ranks: base: 2 self: 2 top: active: true title: value: >- Remplir les informations de la catégorie créée heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une nouvelle actualité : le nom de la catégorie, un résumé, et éventuellement une catégorie parente si nécessaire. Vous pouvez également choisir une image pour illustrer votre catégorie.

notes: >- image: id: "3beb3ac7-a3c9-4921-8e09-69c883199618" file: "3beb3ac7-a3c9-4921-8e09-69c883199618" alt: >- credit: >- - kind: block template: title title: >- Relier une actualité à une ou plusieurs catégories slug: >- relier-une-actualite-a-une-ou-plusieurs-categories ranks: base: 2 self: 2 top: active: true title: value: >- Relier une actualité à une ou plusieurs catégories heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >-

Pour relier une actualité à une catégorie, il faut tout d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'actualité et cochez la case correspondante à la ou aux catégories souhaitées.

notes: >- image: id: "56baecf5-4f4b-47f3-9fc9-71bf68e79aa7" file: "56baecf5-4f4b-47f3-9fc9-71bf68e79aa7" alt: >- credit: >- - kind: block template: title title: >- Gérer les catégories slug: >- gerer-les-categories ranks: base: 2 self: 2 top: active: true title: value: >- Gérer les catégories heading: 2 data: layout: classic - kind: block template: chapter title: >- slug: >- ranks: base: 3 top: active: false data: layout: no_background text: >- notes: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à toutes les actualités de la catégorie sélectionnée.

image: id: "2ca92ecd-3bf7-4da5-8b03-4a4fe9337c1e" file: "2ca92ecd-3bf7-4da5-8b03-4a4fe9337c1e" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: base: 3 top: active: false data: description: >- layout: grid images: - id: "0d449ed6-e983-4737-bb9e-4fcb381e73d4" file: "0d449ed6-e983-4737-bb9e-4fcb381e73d4" alt: >- credit: >- text: >-

En cliquant sur le nom de la catégorie, vous accédez aux actualités liées à cette catégorie. Vous avez la possibilité de les modifier directement, ainsi que les informations de la catégorie en cliquant sur "Modifier" en bas à droite de la page.

- id: "73ef9d9f-8e71-4477-ab15-d7d40ec7a960" file: "73ef9d9f-8e71-4477-ab15-d7d40ec7a960" alt: >- credit: >- text: >-

En cliquant sur le bouton "Modifier", vous accédez aux informations essentielles de la catégorie sélectionnée.
