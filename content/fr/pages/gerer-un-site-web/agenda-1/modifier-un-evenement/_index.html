Comment modifier un événement ?

body: >-


Vous pouvez modifier les blocs de contenu de votre événement ainsi que les informations de votre événement. Ces actions sont séparées. Pour modifier vos événements, cliquez sur l'onglet "Agenda". Ensuite, vous pouvez accéder à l'événement souhaité en cliquant sur le nom de l'événement.

Modifier les informations d'un événement

Vous êtes sur l'événement sélectionné pour le modifier.

Si vous souhaitez modifier les informations de l'événement, vous devez cliquer sur le bouton "Modifier" en haut à droite

Vous avez accès aux informations de l'événement que vous pouvez modifier.

Modifier les blocs de contenu d'un événement

Vous êtes sur l'événement sélectionné pour modifier les blocs de contenus.

En cliquant sur modifier sur le bloc choisi, vous accédez au bloc pour l'éditer.

Vous pouvez modifier les informations. Vous devez cliquer sur "enregistrer" pour valider les modifications.
