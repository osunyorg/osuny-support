Comment créer un événement parent ?

body: >-


Vous pouvez utiliser l'agenda pour créer un ou plusieurs événements"enfant" associé à un événement "parent", par exemple un concert dans un festival.

Ajouter un événement enfant

Vous pouvez dans l'événement "parent" ajouter un ou plusieurs événements "enfant", si l'événement est sur une période supérieure à un jour ou si plusieurs événements sont prévus sur une même journée. 

Vous pouvez rentrer les informations de l'événement créé. Il faut que la date de l'événement "enfant" soit comprise dans la plage définie dans l'événement "parent".

Vous associez ici l'événement parent.

Vous pouvez rajouter un créneau horaire pour l'événement enfant

Vous pouvez choisir l'heure, la durée et le lieu de l'événement.

Le créneau horaire est complété.

Événements "enfant" et "événement "parent"

L'événement "enfant" apparait dans la fiche de l'événement "parent". 

Vous pouvez rajouter autant d'événement "enfant" que vous le souhaitez.
