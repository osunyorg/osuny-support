Comment ajouter un agenda ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 26 text: >- moins d'une minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

La fonctionnalité Agenda vous permet d'ajouter des événements avec une date de début et de fin, ainsi que des horaires précis. Les événements passés seront automatiquement archivés et resteront toujours accessibles.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Ajouter un agenda slug: >- ajouter-un-agenda ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "6abab2bc-0444-46e7-9442-8de2783976db" file: "6abab2bc-0444-46e7-9442-8de2783976db" alt: >- credit: >- text: >-

Pour ajouter la fonctionnalité Agenda, vous devez cliquer sur "modifier" quand vous êtes sur la page d'accueil de votre site.

- id: "3403bbab-a93d-4652-9e00-372fc961c415" file: "3403bbab-a93d-4652-9e00-372fc961c415" alt: >- credit: >- text: >-

Ensuite vous cochez la case "Agenda" pour l'activer et qu'il apparaisse ensuite dans votre menu sur le côté.
