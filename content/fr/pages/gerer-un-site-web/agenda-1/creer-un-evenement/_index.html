Comment créer un événement ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 125 text: >- 2 minutes contents: - kind: block template: title title: >- Créer un événement slug: >- creer-un-evenement ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Pour créer un nouvel événement, vous devez cliquer sur l'onglet "Agenda", puis sur sur le bouton "Créer" en haut au-dessus de la liste des événements.

image: id: "5291e9eb-ad41-49ba-b370-8dbba4ac4910" file: "5291e9eb-ad41-49ba-b370-8dbba4ac4910" alt: >- credit: >- - kind: block template: title title: >- Indiquer les informations de l'événement créé slug: >- indiquer-les-informations-de-l-evenement-cree ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Il s'agit des informations à remplir lors de la création d'un événement. La partie contenu : le titre de votre événement, un sous-titre, un résumé. Les informations concernant la date de l'événement, le début, la fin, les horaires

Vous pouvez choisir de publier ou non la page en cochant la case correspondante.

Vous pouvez également décider de lier votre page à une catégorie si vous avez déjà créé des catégories, comme expliqué dans la partie "Gestion des catégories" des événements.

Ajoutez une image d'illustration à la page en choisissant un fichier depuis votre ordinateur ou votre mobile.

Il est possible de recadrer les images téléchargées depuis son ordinateur comme on le souhaite.

- id: "7b671546-2950-402a-be0d-c20638a504ca" file: "7b671546-2950-402a-be0d-c20638a504ca" alt: >- credit: >- text: >-

L'image est insérée. Vous devez manuellement indiquer le crédit de l'image.

- id: "cc22a1eb-5d2c-49b7-94c3-4aac587f2ce2" file: "cc22a1eb-5d2c-49b7-94c3-4aac587f2ce2" alt: >- credit: >- text: >-

Ajoutez une image d'illustration à la page en choisissant une image parmi les bases de données d'images libres de droits d'Unsplash ou de Pexels.

- id: "4f5c4204-dd64-4246-a1b2-2b2c2061eeea" file: "4f5c4204-dd64-4246-a1b2-2b2c2061eeea" alt: >- credit: >- text: >-

En cliquant sur "Chercher une image", vous accédez aux bases de données d'images libres de droits Unsplash et Pexels. Vous entrez un mot-clé et des images vous sont proposées.

- id: "34b71d74-1555-4cbf-9aa7-c7d1f6f27c17" file: "34b71d74-1555-4cbf-9aa7-c7d1f6f27c17" alt: >- credit: >- text: >-

Une fois que vous avez sélectionné l'image sur Unsplash ou Pexels, elle sera insérée et le crédit sera automatiquement ajouté.

Vous pouvez composer votre page à l'aide d'un système de blocs de contenus. Ces blocs vous permettent d'organiser votre page comme vous le souhaitez. Vous pouvez utiliser des blocs de contenu de base tels que des blocs de texte, d'image, de vidéo ou de son, des blocs narratifs avec des galeries de photos, des témoignages ou encore des appels à l'action, des blocs de listes pour faire des liens entre vos pages, ainsi que des techniques pour insérer une page de contact ou encore du code HTML.

Vous pouvez également rajouter des titres et les associer ou non au bloc de contenu souhaité. Cette gestion des titres et des blocs de contenus va vous permettre d'obtenir une composition de vos pages simple et conviviale.

Retrouver l'explication de tous ces blocs de contenus dans la rubrique "Composer avec les blocs - lien externe".

Une fois les blocs de contenu créés, votre événement est organisé comme vous pouvez le voir sur la capture d'écran. Vous pouvez déplacer les différents blocs et titres comme vous le souhaitez et réorganiser votre page.
