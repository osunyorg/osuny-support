Vous avez la possibilité de créer des catégories pour vos événements, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux événements d'une catégorie donnée. Un événement peut être classé dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: heading title: >- Créer une catégorie slug: "creer-une-categorie" position: 1 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Agenda", puis sur "Créer" à côté de l'espace Catégories.

image: id: "9947c715-4959-4557-a9b6-96eaa238965a" file: "9947c715-4959-4557-a9b6-96eaa238965a" alt: >- credit: >- - kind: heading title: >- Remplir les informations de la catégorie créée slug: "remplir-les-informations-de-la-categorie-creee" position: 2 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir une image.

image: id: "856536b7-ef7c-48e8-8192-c1ca7089cc5d" file: "856536b7-ef7c-48e8-8192-c1ca7089cc5d" alt: >- credit: >- - kind: heading title: >- Relier un événement à une catégorie slug: "relier-un-evenement-a-une-categorie" position: 3 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Pour associer un événement à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'événement et sélectionnez la catégorie souhaitée.

image: id: "5a1a1f58-decf-4dd7-8eb1-4ac45b5149a3" file: "5a1a1f58-decf-4dd7-8eb1-4ac45b5149a3" alt: >- credit: >- - kind: heading title: >- Gérer les catégories slug: "gerer-les-categories" position: 4 rank: 2 - kind: block template: image title: >- ranks: self: false data: text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les événements de la catégorie sélectionnée.
