Comment créer une catégorie ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 72 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Vous avez la possibilité de créer des catégories pour vos événements, afin de les classer par thème. Cette fonctionnalité permet d'accéder facilement aux événements d'une catégorie donnée. Un événement peut être classé dans plusieurs catégories.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une catégorie slug: >- creer-une-categorie ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Pour créer une catégorie, vous devez cliquez sur l'onglet "Agenda", puis sur "Créer" à côté de l'espace Catégories.

image: id: "9947c715-4959-4557-a9b6-96eaa238965a" file: "9947c715-4959-4557-a9b6-96eaa238965a" alt: >- credit: >- - kind: block template: title title: >- Remplir les informations de la catégorie créée slug: >- remplir-les-informations-de-la-categorie-creee ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Vous pouvez entrer ici le nom de la catégorie et choisir une image.

image: id: "856536b7-ef7c-48e8-8192-c1ca7089cc5d" file: "856536b7-ef7c-48e8-8192-c1ca7089cc5d" alt: >- credit: >- - kind: block template: title title: >- Relier un événement à une catégorie slug: >- relier-un-evenement-a-une-categorie ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Pour associer un événement à une catégorie, il faut d'abord créer les catégories. Ensuite, allez sur la page de modification des informations de l'événement et sélectionnez la catégorie souhaitée.

image: id: "5a1a1f58-decf-4dd7-8eb1-4ac45b5149a3" file: "5a1a1f58-decf-4dd7-8eb1-4ac45b5149a3" alt: >- credit: >- - kind: block template: title title: >- Gérer les catégories slug: >- gerer-les-categories ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Une fois les catégories créées, vous pouvez les organiser comme vous le voulez, modifier les informations et accéder à tous les événements de la catégorie sélectionnée.

image: id: "d540e5d1-128a-419d-a2c7-7d7505e7001b" file: "d540e5d1-128a-419d-a2c7-7d7505e7001b" alt: >- credit: >- - kind: block template: gallery title: >- slug: >- ranks: self: 2 data: description: >- layout: grid images: - id: "d88d17d9-26dd-499f-a284-0a6c53fba52e" file: "d88d17d9-26dd-499f-a284-0a6c53fba52e" alt: >- credit: >- text: >-

En cliquant sur "voir", vous accédez aux blocs de contenus de la catégorie sélectionnée.

- id: "151619e3-931a-40c6-96e8-ff8487765b44" file: "151619e3-931a-40c6-96e8-ff8487765b44" alt: >- credit: >- text: >-

Vous pouvez ajouter ou modifier du contenu de la catégorie choisie. Vous pouvez ajouter ou modifier des titres ou des blocs de contenu.

- id: "827b917e-254b-4223-83b3-de5f2741d9f5" file: "827b917e-254b-4223-83b3-de5f2741d9f5" alt: >- credit: >- text: >-

En cliquant sur "modifier", vous accédez aux informations essentielles de la catégorie sélectionnée.

- id: "cc34e4a8-96a5-4bda-85f9-6f1aa21abefd" file: "cc34e4a8-96a5-4bda-85f9-6f1aa21abefd" alt: >- credit: >- text: >-

Vous pouvez modifier le nom, le résumé ou l'image de la catégorie sélectionnée.
