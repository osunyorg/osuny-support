Comment créer une page ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 173 text: >- 3 minutes contents: - kind: block template: chapter title: >- slug: >- ranks: self: false data: layout: alt_background text: >-

Une page vous permet de créer et afficher du contenu sur votre site Web. Vous pouvez créer autant de pages que vous le souhaitez. Après la création d'une page, vous pouvez l'ajouter à votre menu de navigation pour la retrouver facilement.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Créer une page slug: >- creer-une-page ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Pour créer une nouvelle page au site web, vous devez cliquez sur l'onglet "Pages", puis sur "Créer"en haut.

image: id: "dbb7541e-9953-4562-b978-009fdb9b35f4" file: "dbb7541e-9953-4562-b978-009fdb9b35f4" alt: >- credit: >- - kind: block template: title title: >- Indiquer les informations de la page créée slug: >- indiquer-les-informations-de-la-page-creee ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: false data: text: >-

Il s'agit des informations essentielles à remplir lors de la création d'une nouvelle page : le titre de votre page, un résumé, et éventuellement un nom différent pour votre fil d'Ariane si le titre est très long par exemple.

Vous pouvez choisir de publier ou non la page en cochant la case correspondante.

Vous pouvez également décider de lier votre page à une page parente choisie, comme expliqué dans la partie Arborescence - lien externe de ce site support.

image: id: "d3b0be96-7b73-4fe3-a4ac-86a3aa590912" file: "d3b0be96-7b73-4fe3-a4ac-86a3aa590912" alt: >- credit: >- - kind: block template: title title: >- Choisir le format de la page slug: >- choisir-le-format-de-la-page ranks: self: 2 data: layout: classic - kind: block template: gallery title: >- slug: >- ranks: self: false data: description: >-

Vous avez la possibilité de publier votre page en pleine largeur, ce qui changera l'aspect visuel de votre page. Sur de grands écrans, la page en pleine largeur utilisera tout l'espace disponible, ce qui est pertinent pour événementialiser une page.

Si la page n'est pas en pleine largeur, l'espace dédié au contenu sera réduit pour faciliter la lecture, et l'espace libre pourra être utilisé pour une table des matières facilitant la navigation.

Vous avez la possibilité d'illustrer votre page avec une image en utilisant une image de votre ordinateur ou mobile, ou en utilisant nos bases de données d'images libres de droits intégrées à Osuny, Unsplash ou Pexels.

Ajoutez une image d'illustration à la page en choisissant un fichier depuis votre ordinateur ou votre mobile.

Il est possible de recadrer les images téléchargées depuis son ordinateur comme on le souhaite.

L'image est insérée. Vous devez manuellement indiquer le crédit de l'image.

- id: "099fa069-56a7-4abd-90aa-2c5ac69ee608" file: "099fa069-56a7-4abd-90aa-2c5ac69ee608" alt: >- Focus sur "Chercher une image" sur des sites libres de droits Unsplash et Pexels, intégrés au Back Office credit: >- text: >-

Ajoutez une image d'illustration à la page en choisissant une image parmi les bases de données d'images libres de droits d'Unsplash ou de Pexels.

- id: "87d18b04-7d78-42c4-9556-28d2ab49557a" file: "87d18b04-7d78-42c4-9556-28d2ab49557a" alt: >- credit: >- text: >-

En cliquant sur "Chercher une image", vous accédez aux bases de données d'images libres de droits Unsplash et Pexels. Vous entrez un mot-clé et des images vous sont proposées.

- id: "6f96bef4-31ce-464d-af8d-38ece433574d" file: "6f96bef4-31ce-464d-af8d-38ece433574d" alt: >- credit: >- text: >-

Une fois que vous avez sélectionné l'image sur Unsplash ou Pexels, elle sera insérée et le crédit sera automatiquement ajouté.

Vous pouvez composer votre page à l'aide d'un système de blocs de contenus. Ces blocs vous permettent d'organiser votre page comme vous le souhaitez. Vous pouvez utiliser des blocs de contenu de base tels que des blocs de texte, d'image, de vidéo ou de son, des blocs narratifs avec des galeries de photos, des témoignages ou encore des appels à l'action, des blocs de listes pour faire des liens entre vos pages, ainsi que des techniques pour insérer une page de contact ou encore du code HTML.

Vous pouvez également rajouter des titres et les associer ou non au bloc de contenu souhaité. Cette gestion des titres et des blocs de contenus va vous permettre d'obtenir une composition de vos pages simple et conviviale.

Retrouver l'explication de tous ces blocs de contenus dans la rubrique "Composer avec les blocs - lien externe".

Une fois les blocs de contenu créés, votre page est organisée comme vous pouvez le voir sur la capture d'écran. Vous pouvez déplacer les différents blocs et titres comme vous le souhaitez et réorganiser votre page.
