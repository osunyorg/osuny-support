Que faire avant la publication d'une page ?

header_text: >- header_cta: display: false label: >- target: "" external: false contents_reading_time: seconds: 43 text: >- 1 minute contents: - kind: block template: chapter title: >- slug: >- ranks: self: 2 data: layout: alt_background text: >-

Vous pouvez décider de publier directement une page au moment de sa création - lien externe, en cochant la case "Publié" prévue à cet effet. Cependant, avant de publier, vous avez la possibilité de prévisualiser le contenu et de vérifier l'accessibilité de vos contenus.

notes: >- alt: >- credit: >- - kind: block template: title title: >- Vérifier l'accessibilité des contenus slug: >- verifier-l-accessibilite-des-contenus ranks: self: 2 data: layout: classic - kind: block template: image title: >- slug: >- ranks: self: 2 data: text: >-

Sur la page créée, vous trouverez un récapitulatif de l'accessibilité de vos blocs de contenu. Cela vous permettra d'accéder directement aux blocs qui ne sont pas accessibles.
