Comment faire des modifications dans une offre d'emploi ?

body: >-


Vous pouvez modifier les blocs de contenu de votre offre d'emploi ainsi que les informations de votre offre d'emploi. Ces actions sont séparées. Pour modifier vos offres d'emploi, cliquez sur l'onglet "offre d'emploi". Ensuite, vous pouvez accéder à l'offre d'emploi souhaitée en cliquant sur le nom de offre d'emploi.

Modifier les informations d'une offre d'emploi

Vous êtes sur l'offre d'emploi sélectionnée pour la modifier.

Si vous souhaitez modifier les informations de l'offre d'emploi, vous devez cliquer sur le bouton "Modifier" en haut à droite.

Vous avez accès aux informations de l'offre d'emploi que vous pouvez modifier.

Modifier les blocs de contenu d'une offre d'emploi

Vous êtes sur l'offre d'emploi sélectionnée pour la modifier.

Vous avez accès aux blocs de contenu de l'offre d'emploi que vous pouvez modifier. En cliquant sur "modifier" à côté du bloc de contenu, vous accédez au bloc à modifier.

Vous pouvez modifier le bloc de contenu sélectionné.
