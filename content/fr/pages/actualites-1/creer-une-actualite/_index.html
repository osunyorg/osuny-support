Comment créer une actualité ?

body: >-

Créer une actualité

Pour créer une actualité, l'utilisateur doit cliquer sur le bouton
"créer" en bas à droite ou "nouvelle curation" s'il s'agit d'un flux
RSS.

Remplir une actualité

Au clic sur "créer", l'utilisateur arrive sur une page vide pour créer une actualité avec les champs suivants : 
Contenu :  Titre (obligatoire) / Chapô
Catégorie : Choix d'une ou plusieurs catégories définies auparavant
SEO :  Meta Description
Informations : Publié ou non / Mis en avant ou non / Date de publication / Auteur.rice / Slug
Image :  Choisir un fichier / Texte alternatif / Crédit / Chercher une image (Unsplash ou Pexels)
Il clique sur enregistrer pour valider l'actualité. Possibilité de choisir de publier la page ou non et donc de la mettre en brouillon. Une
fois l'actualité créée, l'utilisateur peut remplir l'actualité par les
blocs de contenus en cliquant sur la fiche de l'actualité, sur son nom.

Prévisualiser une actualité

L'utilisateur peut prévisualiser une actualité sous mobile, tablette ou desktop.

Ajouter une URL

Au clic sur "nouvelle curation", l'utilisateur a la possibilité
d'ajouter une URL dans le but de récupérer les nouvelles actualités
automatiquement via un flux RSS.
